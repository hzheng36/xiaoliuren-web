小六壬网页版 v8.1 场景规则增强版
=================================

基于：v8 场景化问事断法版（含清空事项按钮）

本版更新：
- 增强本地问事规则库：新增事业/工作/项目细分断法，并增强天气、感情、财运、失物、消息、合作、考试、健康等专项提示。
- 问事专项解读增加“专项细分提示”，会根据用户填写内容进一步补充，例如下雨/暴雨/出行天气、前任/复合、投资/回款、金属失物等。
- 新增“复制结果”按钮，可一键复制占问事项、起课方式、取数、落宫、综合判断、问事专项解读、三象和行动建议。
- 不联网、不接 API，全部分析仍在 index.html 本地完成。
- 起课算法、农历取数、连续顺推落宫和整体排版保持不变。

使用：
- 直接打开 index.html。
- 或上传 index.html 覆盖 GitHub Pages 仓库根目录中的旧文件。
