## v10.5.209 问什么答什么白话总结与复盘回填合并增强版

这一版继续按照合并版节奏推进，重点不是再加一堆新面板，而是把普通问事结果页做得更像“直接回答用户自己写的那一句问题”。本版把场景、占问事项、三象结果、日常建议和后续复盘草稿进一步串起来。

### 本版新增 / 修复

1. 问什么答什么白话总结
   - 普通问事结果页新增“问什么答什么 · 白话总结”面板；
   - 根据用户填写的占问事项识别“能不能 / 要不要 / 什么时候 / 在哪里 / 对方态度”等问法；
   - 直接用日常语言说明这件事该怎么理解，不只堆宫位术语。

2. 场景专项建议增强
   - 按当前场景给出三条更容易执行的建议；
   - 结构固定为“现在先做 / 接下来观察 / 以后复盘”；
   - 感情、财运、事业、健康、失物、出行、合同、合作、车辆等场景都有更贴近日常的提示。

3. 占问事项与结果联动增强
   - 把“你问的重点、放到场景里看、本课落点”放在同一个面板里；
   - 三象结果会以“本象 → 中象 → 末象”的生活线索解释；
   - 让用户更容易理解这课到底是在回答什么。

4. 复盘回填草稿
   - 新增“复制复盘草稿”；
   - 新增“保存复盘草稿”；
   - 新增“打开复盘并带入草稿”；
   - 草稿只会填入复盘备注、学习标签和学习要点，不会自动保存，也不会修改起课数字、落宫或断卦结论。

5. 暗夜 / 阅读模式同步适配
   - 新增白话总结面板、建议卡片、复盘草稿按钮均已补暗夜样式；
   - 继续避免浅灰底、白块、浅紫字和低对比文字问题。

6. Debug / 发版体检增强
   - 新增 `xlr209AnswerReviewAudit()`；
   - 新增 `question-answer-review-v209` 与 `new-content-night-theme-v209` 发版体检项；
   - 主线 source 统一到 `mainline-v10.5.209`。
