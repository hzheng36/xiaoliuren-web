# 小六壬个人占卜台 · v9.9.16 设置增强与图标版

这是一个纯前端小六壬个人网页工具，适合部署在 GitHub Pages。历史、复盘、模板和设置均保存在当前浏览器本地，不联网、不上传数据。

## 本版更新

- 默认字体恢复到原来的精致比例。
- 保留「阅读文字大小」设置，用户可自行调节结果和说明文字大小。
- 设置中新增：显示快捷按钮、显示章节卡点导航、显示阅读进度条、默认客户端模式。
- 设置中新增「数据管理」入口，可快速打开历史记录、导出/导入完整备份、恢复默认设置。
- 新增 Home Screen 图标：iPhone / iPad 通过 Safari 添加到主屏幕时，会显示定制小六壬图标。

## 起课规则

本工具采用 **三数连续顺推法**：

1. 第一数从大安起数。
2. 第二数从第一数落宫处继续顺推。
3. 第三数从第二数落宫处继续顺推。

六宫顺序：

```text
大安 → 留连 → 速喜 → 赤口 → 小吉 → 空亡
```

## 主要功能

- 数字起课、时辰起课、事件起课
- 场景化问事断法与本地规则库
- 简洁版 / 详细版结果阅读
- A/B 选择对比
- 历史记录、历史分析、复盘时间线
- TXT 导出、历史 JSON、模板 JSON、完整备份 JSON
- 问题模板、自定义模板、模板收藏与批量管理
- 场景规则查询
- 阅读设置、章节卡点导航、阅读进度条、快捷按钮

## 本地数据说明

历史记录、复盘、模板、阅读设置等均保存在当前浏览器本地。换设备或清理缓存前，建议先导出 **完整备份 JSON**。

## 导入导出

- TXT：用于阅读、分享、给后续分析参考。
- 历史 JSON：只迁移历史记录、收藏、复盘和 A/B 数据。
- 模板 JSON：只迁移问题模板。
- 完整备份 JSON：迁移整个个人占卜台，包括历史、复盘、模板和设置。

## 部署到 GitHub Pages

将以下文件放到仓库根目录：

```text
index.html
README.md
apple-touch-icon.png
icon-192.png
icon-512.png
manifest.webmanifest
```

然后在 GitHub 仓库 Settings → Pages 中选择 main / root 发布。

## iPhone / iPad 添加到主屏幕

1. 用 Safari 打开 GitHub Pages 网址。
2. 点击分享按钮。
3. 选择「添加到主屏幕」。
4. 图标会使用本版提供的 `apple-touch-icon.png`。
