# 小六壬占卜 · v10.5.210 历史暗夜回看修复、按钮状态统一与全局折叠合并版

## v10.5.210 历史暗夜回看修复、按钮状态统一与全局折叠合并版

这一版按你反馈的问题做合并修复，不继续堆新功能。重点修复 v10.5.209 在暗夜模式、历史回看、按钮状态和折叠控制上的几个实际体验问题。

### 本版新增 / 修复

1. 暗夜模式历史记录与历史回看修复
   - 修复暗夜模式下查看历史记录或历史回看结果时，列表、卡片、文字可能因为颜色继承或覆盖顺序导致看起来空白的问题；
   - 对历史列表、时间线、历史卡片、筛选栏、批量工具和历史回看结果页补强暗夜可读性兜底。

2. 重点阅读模式排版修复
   - 修复“当前为重点阅读模式”提示沿用 `.card::after` 绝对定位后变成大块遮挡的问题；
   - 历史回看占卜结果时，结果标题、起课信息、按钮和普通问事快速读法不再被红色/粉色块压住。

3. 按钮高亮状态统一
   - 统一历史记录顶部的“全部/只看收藏”、“列表/时间线”、“待复盘/全部复盘状态”、“全部案例/高价值/已验证/待复盘案例”等按钮高亮逻辑；
   - 当前真正生效的筛选状态会保持高亮，不生效的按钮不再误高亮。

4. 历史记录 / 历史分析增加总开关
   - 历史记录弹窗顶部新增“全部展开 / 全部折叠”；
   - 历史分析弹窗顶部新增“全部展开 / 全部折叠”；
   - 只控制界面展开状态，不修改历史数据、复盘数据、导出数据、起课数字、落宫或断卦结论。

5. Debug warning 清理
   - 清理 `done`、`latestReviewEntry`、`q` 重复 function 定义 warning；
   - 新增 `xlr210FixAudit()`，用于检查暗夜历史修复、结果页排版修复、按钮状态同步和全局折叠开关是否挂载。
