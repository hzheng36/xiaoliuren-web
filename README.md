# 小六壬占卜 · v10.5.203 历史中心分组导航与合并升级节奏版

## v10.5.203 历史中心分组导航与合并升级节奏版

这一版承接 v10.5.202 的“历史中心精简视图默认化与面板折叠记忆版”，开始按更快的“合并版”节奏推进：不再为了每个小按钮单独开一版，而是把同一条主线下的导航、体验、暗夜/阅读模式、Debug 与版本源检查合并交付。

### 本版新增 / 修复

1. 历史中心面板分组导航
   - 新增“历史中心面板分组导航与合并升级节奏”面板；
   - 将历史中心大量面板整理成常用面板、复盘闭环、案例库、导出报告、体检面板、历史列表和底部导出；
   - 支持一键跳转到对应区域，减少在历史中心长页面中来回找面板。

2. 快捷定位
   - “常用面板”定位折叠记忆 / 搜索记忆 / 批量保护等常用入口；
   - “复盘闭环”定位字段补全、复盘趋势、复盘模板和学习闭环；
   - “案例库”定位历史案例库、经验库和章节摘要；
   - “导出报告”定位质量仪表盘、打印字段、报告索引和 PDF / CSV 导出；
   - “体检面板”定位稳定基准、移动端、性能、暗夜 / 阅读模式和 Debug 总检。

3. 合并升级节奏说明
   - 后续每版尽量合并 2-4 个相关小更新；
   - 每版固定同步：功能、体验、暗夜/阅读模式、移动端、Debug、版本源、README、manifest、service-worker；
   - 历史中心后续少堆碎片面板，优先做阶段收口或转向更高价值主线。

4. 暗夜 / 阅读模式同步适配
   - 新增分组导航面板、KPI 卡片、入口卡片、标签、工具栏和说明区域均已补暗夜样式；
   - 继续防止新增内容出现浅灰底、白块、浅紫字和低对比文字。

5. Debug / 发版体检增强
   - 新增 `history-group-nav-v203` 发版体检项；
   - 新增 `new-content-night-theme-v203` 暗夜/阅读模式体检项；
   - Debug 历史模块快照新增 `groupNavAudit`。

6. 版本源同步
   - 页面 title、meta、首页版本、版本内容更新、README、manifest、service-worker 已同步到 v10.5.203；
   - 主线 source 已统一到 `mainline-v10.5.203`；
   - service worker cache 已更新为 `xiaoliuren-v10.5.203`。

### 本版检查

- 已检查主脚本语法。
- 已检查 manifest JSON。
- 已检查重复 ID。
- 已检查旧版本 source 残留。
- 已检查 PDF 旧版本硬编码残留。
- 已检查 ZIP 完整性。
- 本版只优化历史中心导航、面板定位、合并升级节奏、Debug 体检和暗夜/阅读模式，不修改核心起课算法、落宫规则、断卦逻辑和 JSON 知识库。

### 下一版建议

v10.5.204 建议开始做“合并版主线推进”：一次合并普通问事结果页体验、场景知识库显示、移动端阅读和 Debug 检查，不再只围绕历史导出/历史分析继续单点升级。
