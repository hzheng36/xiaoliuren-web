# 小六壬占卜 · v10.5.184 复盘编辑与案例学习增强版

## v10.5.184 复盘编辑与案例学习增强版

这一版承接 v10.5.179 的标签/备注/归档、v10.5.180 的复盘时间线、v10.5.181 的标签洞察、v10.5.182 的历史案例库，以及 v10.5.183 的导出联动，把“复盘”继续升级成可长期学习的案例沉淀系统。

### 本版新增

1. 复盘弹窗新增学习字段
   - 新增“原因归类”：可标记基本应验、应期提前、应期延后、场景选错、问题不够具体、落宫解读偏差、外部条件变化、人为行动改变结果、样本不足/结果不明；
   - 新增“案例学习标签”：例如应期提前、三象应验、场景选错、值得复查；
   - 新增“案例学习要点”：记录这条案例对下次判断的启发。

2. 复盘时间线增强
   - 时间线中同步显示原因归类、学习标签和学习要点；
   - 编辑旧复盘时会完整带回准确度、实际结果、备注、原因归类、学习标签和学习要点；
   - 删除、清空、追加复盘仍保持原来的多条复盘逻辑。

3. 历史搜索与导出扩展
   - 历史搜索范围新增原因归类、学习标签和学习要点；
   - 历史 CSV 新增最新原因归类、最新学习标签、最新学习要点；
   - 复盘明细 CSV 新增原因归类、学习标签、学习要点；
   - TXT 复盘导出同步输出学习字段。

4. 新增“复盘原因与案例学习”洞察面板
   - 历史记录中心新增学习洞察面板；
   - 自动统计当前筛选范围内的复盘条数、已归类复盘、学习标签数量、学习要点数量；
   - 显示偏差原因排行和学习标签排行；
   - 点击原因或标签可写入历史搜索框并联动筛选。

5. 案例库交互补齐
   - 补齐高价值案例/已验证/待复盘案例筛选按钮事件；
   - 补齐案例排序下拉事件；
   - 补齐案例库摘要卡片中的“查看”和“标记高价值”按钮事件。

6. Debug / 发版体检增强
   - 新增 `history-review-learning-v184` 发版体检项；
   - 检查复盘学习洞察面板、KPI、列表、刷新按钮、统计函数和搜索联动函数是否挂载；
   - 继续检查历史案例库、导出联动、版本源和主线 source。

### 本版检查

- 已统一主线 source 标记为 `mainline-v10.5.184`。
