# 小六壬个人占卜台 · v9.9.18 白话答案增强版

这是一个纯前端小六壬个人网页工具，适合部署在 GitHub Pages。历史、复盘、模板和设置均保存在当前浏览器本地，不联网、不上传数据。

## 本版更新

- 新增 PWA 离线缓存：添加到主屏幕后体验更接近 App，网络不好时也可打开已缓存的工具页面。
- 答案摘要升级为更直观的结构：一句话直断、偏向判断、结果节奏、通俗解释、行动建议和注意事项。
- 白话表达更贴近用户填写的占问事项，减少术语门槛，让普通用户更快理解结果。
- 继续保留详细版中的术语、三象解读、单宫/双宫断法、象意补充和雷达分析。
- 同步完善 `manifest.webmanifest`，新增 `service-worker.js`。

## 起课规则

本工具采用 **三数连续顺推法**：

1. 第一数从大安起数。
2. 第二数从第一数落宫处继续顺推。
3. 第三数从第二数落宫处继续顺推。

六宫顺序：

```text
大安 → 留连 → 速喜 → 赤口 → 小吉 → 空亡
```

## 主要功能

- 数字起课、时辰起课、事件起课
- 场景化问事断法与本地规则库
- 简洁版 / 详细版结果阅读
- A/B 选择对比
- 历史记录、历史分析、复盘时间线
- TXT 导出、历史 JSON、模板 JSON、完整备份 JSON
- 问题模板、自定义模板、模板收藏与批量管理
- 场景规则查询
- 阅读设置、章节卡点导航、阅读进度条、快捷按钮
- PWA 离线缓存与 Home Screen 图标

## 本地数据说明

历史记录、复盘、模板、阅读设置等均保存在当前浏览器本地。换设备或清理缓存前，建议先导出 **完整备份 JSON**。

## 导入导出

- TXT：用于阅读、分享、给后续分析参考。
- 历史 JSON：只迁移历史记录、收藏、复盘和 A/B 数据。
- 模板 JSON：只迁移问题模板。
- 完整备份 JSON：迁移整个个人占卜台，包括历史、复盘、模板和设置。

## PWA 离线缓存说明

本版新增 `service-worker.js`。首次通过 GitHub Pages 正常访问后，浏览器会缓存核心文件。之后在网络较差或离线时，已添加到主屏幕的工具仍可打开已缓存版本。

如果上传新版本后手机仍显示旧内容，可以尝试：

1. 在 Safari 中刷新网页。
2. 关闭后重新打开主屏幕图标。
3. 必要时在 iPhone 设置中清理 Safari 网站数据后重新添加。

## 部署到 GitHub Pages

将以下文件放到仓库根目录：

```text
index.html
README.md
manifest.webmanifest
service-worker.js
apple-touch-icon.png
icon-192.png
icon-512.png
```

然后在 GitHub 仓库 Settings → Pages 中选择 main / root 发布。

## iPhone / iPad 添加到主屏幕

1. 用 Safari 打开 GitHub Pages 网址。
2. 点击分享按钮。
3. 选择「添加到主屏幕」。
4. 图标会使用 `apple-touch-icon.png`。
