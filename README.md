# 小六壬占卜 · v10.5.185 复盘偏差趋势与案例评分增强版

## v10.5.185 复盘偏差趋势与案例评分增强版

这一版承接 v10.5.184 的原因归类、学习标签和案例学习要点，把复盘学习继续升级成“趋势判断 + 偏差定位 + 案例评分”的长期案例库能力。

### 本版新增

1. 新增“复盘偏差趋势与案例评分”面板
   - 历史记录中心新增独立面板；
   - 自动统计当前筛选范围内的复盘数量、有效复盘、平均准确率、应期偏差和高分案例；
   - 统计只跟随当前历史记录范围，不影响核心起课和断卦。

2. 月份趋势与应期偏差
   - 按复盘时间/更新时间归入月份；
   - 显示每个月复盘数量、准确率、应期提前、应期延后和主要偏差；
   - 点击月份卡片可写入历史搜索框并联动筛选。

3. 场景偏差排行
   - 按场景统计偏差/需复查数量、偏差率和准确率；
   - 汇总主要偏差方向，例如应期偏差、解读偏差、问题/场景偏差、外部/人为变化；
   - 点击场景偏差卡片可快速筛选同类历史。

4. 自动案例学习评分
   - 新增自动评分函数 `historyCaseLearningScore()`；
   - 综合有效准确率、复盘完整度、原因归类、学习标签、学习要点、高价值和收藏等整理信号；
   - 输出 0-100 分与 S级高价值 / A级可参考 / B级需复查 / C级样本不足 / 待复盘等级；
   - 案例库卡片同步显示学习评分。

5. 排序、搜索与导出增强
   - 案例库排序新增“学习评分优先”；
   - 历史搜索可搜索案例学习评分与评分等级；
   - 历史 CSV、复盘 CSV、XLSX/PDF 数据层增加案例学习评分和案例评分等级字段。

6. Debug / 发版体检增强
   - 新增 `history-review-trend-v185` 发版体检项；
   - 检查趋势面板、KPI、列表、刷新按钮、评分函数、筛选联动函数是否挂载；
   - 继续检查历史案例库、复盘学习、导出联动、版本源和主线 source。

### 本版检查

- 已同步更新页面 title、meta、首页版本标题、版本记录、README、manifest、service-worker。
- 已统一主线 source 标记为 `mainline-v10.5.185`。
- 已检查主脚本语法、重复 ID、旧版本残留和 ZIP 完整性。
- 本版只增强复盘趋势、案例评分、排序、搜索和导出字段，不修改核心起课算法、落宫规则、断卦逻辑和 JSON 知识库。
