# 小六壬占卜 · v10.5.172 PDF/Excel 导出中心整合版

## v10.5.172 PDF/Excel 导出中心整合版

这一版基于 v10.5.171 继续整理“导出层”。重点不是新增复杂算法，而是把历史分析和历史记录里越来越多的 TXT、CSV、XLSX、PDF、JSON、完整备份入口收拢成更清楚的“导出中心”，并用格式卡片说明每种导出适合什么用途。

核心原则保持不变：导出功能只读取当前浏览器本地保存的历史、复盘、设置和模板数据；不会联网、不会上传、不会修改起课算法、落宫规则、断卦逻辑、JSON 知识库或复盘结论。

### 本版新增 / 修复

1. 历史分析导出中心：
   - 将 `复盘 TXT`、`历史 TXT`、`历史 CSV`、`复盘 CSV`、`Excel 报表`、`完整 PDF`、`图表 PDF`、`明细 PDF` 整理到统一的格式卡片中。
   - 每个格式卡片说明适合用途，避免用户不知道该导出哪一种。
   - 保留 v10.5.166 之后已经认可的 PDF 视觉风格和 v10.5.163/v10.5.165 的 Excel 图表报表能力。

2. 历史记录导出中心：
   - 统一展示 TXT、CSV、XLSX、PDF、历史 JSON、完整备份 JSON。
   - `历史 JSON` 用于迁移历史记录、收藏、复盘和 A/B 数据。
   - `完整备份 JSON` 用于换设备或清缓存前保存完整数据，包括历史、复盘、A/B、阅读设置、问题模板和模板使用记录。
   - `清空历史` 继续单独放在风险区，避免和导出按钮混在一起误点。

3. 导出格式说明增强：
   - TXT：适合阅读、复制、普通文字保存。
   - CSV：适合 Excel / Numbers / Power BI 结构化分析。
   - XLSX：适合正式看图表总览、统计表、历史明细和复盘趋势。
   - PDF：适合打印、归档、手机阅读和分享。
   - JSON：适合历史迁移。
   - 完整备份 JSON：适合换设备或清缓存前完整保存。

4. Debug / 发版体检增强：
   - 新增 `XLRExportCenterCore` 主命名空间。
   - 新增 `EXPORT_CENTER_MISSING` 检查项。
   - 新增 `PDF/Excel 导出中心整合可用` 发版体检项。
   - Debug 快照会记录导出中心面板、格式卡片、格式覆盖和关键导出按钮是否完整。

5. 继续保持核心隔离：
   - 本版只调整导出入口、说明卡片、CSS 样式和 Debug 审计。
   - 不修改起课算法、落宫规则、历史数据结构、复盘数据和 JSON 知识库断法。

### 使用方法

1. 打开网站。
2. 进入 `历史分析` 或 `历史记录`。
3. 展开或查看 `导出中心`。
4. 按用途选择格式：
   - 只想看文字：TXT。
   - 想拿去 Excel / Power BI 分析：CSV。
   - 想要带图表的 Excel 报表：XLSX。
   - 想打印归档或手机阅读：PDF。
   - 想迁移历史：历史 JSON。
   - 想完整备份设置和模板：完整备份 JSON。
