# 小六壬占卜 · v10.5.202 历史中心精简视图默认化与面板折叠记忆版

## v10.5.202 历史中心精简视图默认化与面板折叠记忆版

这一版承接 v10.5.201 的“稳定基准后问题修复与历史中心体验微调版”，继续减少历史中心打开后的视觉压力：把常用面板默认化、面板开合状态记忆化，并进一步优化手机 / iPad 首次打开体验。

### 本版新增 / 修复

1. 历史中心精简视图与面板折叠记忆
   - 新增“历史中心精简视图与面板折叠记忆”面板；
   - 显示当前视图、可见面板数、展开面板数、折叠记忆状态和暗夜适配状态；
   - 支持保存和恢复当前面板展开/收起状态。

2. 常用面板默认化
   - 历史中心继续优先使用精简视图；
   - 保留常用历史、搜索、批量保护、字段补全、案例库和复查类面板；
   - 高级导出/统计面板仍可通过复盘视图、导出视图或全部展开查看。

3. 折叠状态快捷操作
   - 新增“记住当前开合”；
   - 新增“恢复记忆”；
   - 新增“展开当前视图”；
   - 新增“收起当前视图”；
   - 新增“恢复精简默认”。

4. 手机 / iPad 首次打开体验优化
   - 通过精简视图和折叠记忆减少一次性展开的大量统计面板；
   - 保留 v10.5.195-v10.5.201 的移动端滚动、触控区域和 iPad / Safari 弹窗高度增强；
   - 折叠记忆只影响历史中心 UI，不影响数据和导出范围。

5. Debug / 发版体检增强
   - 新增 `history-fold-memory-v202` 发版体检项；
   - 新增 `new-content-night-theme-v202` 暗夜/阅读模式体检项；
   - Debug 历史模块快照新增 `foldMemoryAudit`。

6. 暗夜 / 阅读模式同步适配
   - 新增折叠记忆面板、KPI 卡片、状态卡片、工具栏、说明区域均已补暗夜样式；
   - 继续防止新增内容出现浅灰底、白块、浅紫字和低对比文字。

7. 版本源同步
   - 页面 title、meta、首页版本、版本内容更新、README、manifest、service-worker 已同步到 v10.5.202；
   - 主线 source 已统一到 `mainline-v10.5.202`；
   - service worker cache 已更新为 `xiaoliuren-v10.5.202`。
