# 小六壬占卜 · v10.5.194 经验库总修复与面板整理版

## v10.5.194 经验库总修复与面板整理版

这一版承接 v10.5.193 的“经验库报告章节摘要与批量复查入口”，按之前约定开始做一次集中整理。重点不是继续堆新面板，而是把 v10.5.179-v10.5.193 连续新增的历史标签、案例库、复盘学习、经验库导出、PDF 目录和章节摘要统一收口，让历史记录中心在桌面端和移动端都更好用，同时继续强化新增内容的暗夜 / 阅读模式检查。

### 本版新增 / 修复

1. 历史经验库面板整理工具栏
   - 历史记录中心新增“面板整理”工具栏；
   - 支持：精简视图、复盘视图、导出视图、全部展开、全部收起；
   - 解决历史记录中心新增面板过多、打开后内容过长的问题；
   - 只整理界面显示，不删除数据，不影响导出和复盘统计。

2. 精简 / 复盘 / 导出三种视图
   - 精简视图：保留标签统计、案例库摘要、快捷复查、经验库和章节摘要；
   - 复盘视图：聚焦复盘原因、偏差趋势、快捷复查、学习闭环和复盘模板；
   - 导出视图：聚焦经验库、质量仪表盘、打印字段、报告索引和章节摘要；
   - 视图选择会保存在当前浏览器本地。

3. 新增面板暗夜 / 阅读模式总修复
   - 统一历史经验库新增面板的背景、边框、summary、按钮、标签和状态提示样式；
   - 继续防止浅灰底、白块残留、浅紫字、低对比文字、按钮/标签没有继承主题变量等问题；
   - 移动端下整理工具栏自动改成更适合手指点击的布局。

4. Debug / 发版体检增强
   - 新增 `history-panel-organizer-v194` 发版体检项；
   - 新增 `new-content-night-theme-v194` 发版体检项；
   - Debug 历史模块快照新增 `panelOrganizerAudit`；
   - 面板整理工具栏、5 个整理按钮、12 个历史经验库面板和暗夜样式都纳入检查。

5. 版本源同步
   - 页面 title、meta、首页版本标题、版本记录、README、manifest、service-worker 已同步到 v10.5.194；
   - 主线 source 已统一到 `mainline-v10.5.194`；
   - service worker cache 已更新为 `xiaoliuren-v10.5.194`。
