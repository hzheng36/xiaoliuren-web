# 小六壬占卜 · v10.5.173 PDF 明细体检与导出范围预览版

## v10.5.173 PDF 明细体检与导出范围预览版

这一版基于 v10.5.172 先修正 PDF 明细模式 Debug 体检误报：明细 PDF 本身已经可以生成，但旧判断把“正文不能有图表总览”简化成“全文不能出现图表总览”，导致封面省略说明提到图表总览时被误判。本版改为正向检查模式标签、明细结构、报告长度和无错误状态；同时继续保留导出中心的导出前范围预览。

核心原则继续保持不变：导出功能只读取当前浏览器本地数据；不会联网、不会上传、不会修改起课算法、落宫规则、断卦逻辑、JSON 知识库、历史记录内容或复盘结论。

### 本版新增 / 修复

1. PDF 明细模式体检修正：
   - 修复 `pdfDetailsOnlyReady: false` 的误判来源。
   - 明细模式允许封面“省略内容”说明中提到图表总览。
   - Debug 只禁止正文真的出现图表章节，例如 `<h2>图表总览</h2>` 或 `chart-section`。
   - 新增 `pdfModeAudit.full/charts/details`，以后能直接看到三种 PDF 模式各自是否通过。

2. 导出前范围预览：
   - 历史分析导出中心顶部新增“导出前范围预览”。
   - 历史记录导出中心顶部新增“导出前范围预览”。
   - 预览会显示历史记录数、复盘明细数、收藏记录数、涉及场景数。

2. 历史分析范围确认：
   - 历史分析侧会明确当前是“全部日期”还是某一天。
   - XLSX 和 PDF 报表继续按历史分析日期筛选生成。
   - 预览区提示 TXT / CSV / JSON 仍按对应导出按钮原逻辑处理，避免用户误解范围。

3. 历史记录范围确认：
   - 历史记录侧预览会跟随搜索词、日期、场景筛选和“只看收藏”刷新。
   - 预览用于导出前核对当前显示范围。
   - 完整备份 JSON 仍会保存所有受保护数据，不会只备份当前筛选结果。

4. Debug / 发版体检增强：
   - `XLRExportCenterCore` 新增范围预览能力标记。
   - `getExportCenterDomStatus()` 会检查导出中心范围预览是否已经渲染。
   - 新增 `EXPORT_SCOPE_PREVIEW_MISSING` 检查项。
   - 新增 `导出中心范围预览与确认可用` 发版体检项。

5. 继续保持导出中心完整性：
   - 保留 v10.5.172 的 TXT / CSV / XLSX / PDF / JSON / 完整备份格式卡片。
   - 保留 v10.5.166 之后用户认可的 PDF 视觉风格。
   - 保留 v10.5.163 / v10.5.165 的 Excel 图表报表能力。

### 使用方法

1. 打开网站。
2. 进入 `历史分析` 或 `历史记录`。
3. 查看 `导出中心` 顶部的“导出前范围预览”。
4. 确认当前范围和数量无误后，再选择导出格式：
   - TXT：阅读、复制、轻量保存。
   - CSV：Excel / Numbers / Power BI 结构化分析。
   - XLSX：Excel 图表报表。
   - PDF：打印、归档、手机阅读。
   - 历史 JSON：迁移历史记录。
   - 完整备份 JSON：换设备或清缓存前完整保存。
