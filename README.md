# 小六壬占卜 · v10.5.179 历史记录归档与标签增强版

## v10.5.179 历史记录归档与标签增强版

这一版继续增强历史记录中心，在 v10.5.178 批量管理基础上加入“标签 / 备注 / 归档”整理能力。适合历史记录越来越多之后，把已处理、长期保留、重点复盘、不同主题的记录分开管理。

### 本版新增

1. 历史标签与备注
   - 每条历史记录可点击“标签/备注”进行整理；
   - 标签支持多个关键词，例如“客户、健康、待复盘”；
   - 备注只用于历史整理，不影响占卜结论、复盘结论和核心断卦逻辑。

2. 归档状态
   - 每条历史记录新增“归档 / 取消归档”；
   - 已归档记录会显示“已归档”标记；
   - 默认历史列表显示未归档记录，减少旧记录干扰。

3. 标签 / 归档筛选
   - 历史记录中心新增标签筛选输入框；
   - 新增归档筛选：未归档、全部含归档、只看归档；
   - 搜索历史时会同时搜索标签与整理备注。

4. 批量整理增强
   - 支持批量加标签；
   - 支持批量移除标签；
   - 支持批量归档；
   - 支持批量取消归档；
   - 继续保留 v10.5.178 的批量收藏、取消收藏、删除、选中导出与撤销保护。

5. CSV / Debug 增强
   - 历史 CSV 与选中 CSV 新增“归档、标签、整理备注”字段；
   - Debug 新增历史标签/归档整理审计；
   - 发版体检新增 `history-tags-archive-v179` 检查项。

- 已检查主脚本语法。
