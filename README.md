## 主要功能

- 数字起课、单数起课、时辰起课、事件起课。
- 命运场景仅使用时辰起盘，适合出生时间参考盘；支持公历/农历与 24 小时制/十二时辰输入。
- 场景化解读：感情、事业、财运、考试、失物、健康、出行、合作、消息等待、天气、求职、家宅、选择、寻人、贵人、运势、命运等。
- 简洁版 / 详细版阅读。
- 历史记录、占例复盘日志、日志状态筛选、重点案例、历史分析。
- TXT / JSON 导入导出与完整备份。
- 问题模板、自定义模板、模板收藏与管理。
- 阅读设置、快捷按钮自定义、章节导航、阅读进度条。

## 起课规则

本工具的主线规则为三数连续顺推法：

1. 第一数从大安起。
2. 第二数从第一数落宫继续起。
3. 第三数从第二数落宫继续起。

命运场景采用：

> 大安起月，月上起日，日上起时。

并以时宫作为自身宫进行命盘排法。

## GitHub Pages / PWA 缓存提示

如果上传新版后仍显示旧版本，通常是浏览器或 PWA Service Worker 缓存导致。可以尝试：

1. 把 ZIP 解压后的 7 个文件全部上传到仓库根目录，尤其不要漏掉 `service-worker.js`。
2. 在 Safari 里直接打开网页地址，刷新两次。
3. iPhone 可进入「设置 → Safari → 高级 → 网站数据」，删除对应站点数据。
4. 如果是主屏幕图标打开仍旧，先删除主屏幕图标，再重新添加。
