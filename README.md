## v10.5.169 PDF 章节分页与页眉页脚增强版

这一版基于 v10.5.168 继续升级 PDF 报告导出层：保留 v10.5.166 已认可的 PDF 视觉风格、v10.5.167 舒展明细卡片和 v10.5.168 三种 PDF 导出模式，不重做整体版式，重点优化打印分页、页眉页脚和后续自定义标题/备注预留。

### 本版新增 / 修复

1. PDF 打印分页增强：
   - 封面后自动进入正文页。
   - 图表总览作为独立章节开始。
   - 历史明细、复盘明细作为独立章节开始。
   - 尽量避免章节标题单独落在页尾。
2. PDF 明细卡片分页保护增强：
   - 历史明细卡片继续使用舒展卡片布局。
   - 复盘明细卡片继续使用舒展卡片布局。
   - 打印时对卡片、字段块、图表块增加 `break-inside: avoid` / `page-break-inside: avoid` 保护。
3. PDF 页眉页脚增强：
   - 打印页眉显示报告标题、统计范围和导出模式。
   - 打印页脚显示本地生成提示和生成时间。
   - 保留浏览器打印设置中的页码开关提示，兼容桌面和手机端保存 PDF。
4. PDF 报告标题/备注预留：
   - 新增 `normalizeAnalyticsPdfReportOptions` 结构。
   - 当前先使用默认标题和自动备注。
   - 后续可以扩展为用户自定义报告标题、导出备注、报告说明，不影响现有按钮。
5. Debug / 发版体检增强：
   - 新增 PDF 分页/页眉页脚/标题备注预留检查。
   - 新增 `ANALYTICS_PDF_PRINT_LAYOUT_MISSING` 警告项。
   - 新增 `PDF 章节分页与页眉页脚检查通过` 通过项。

### 使用方法

1. 打开网站。
2. 进入历史分析或历史记录。
3. 在导出区选择：
   - `导出完整 PDF`
   - `只导出图表 PDF`
   - `只导出明细 PDF`
4. 浏览器会打开打印窗口。
5. 在打印选项中选择 `保存为 PDF`。
