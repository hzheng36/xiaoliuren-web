## v10.5.23 事业模块版
- 命运结果页新增“事业与财富参考”子模块。
- 事业模块沿用健康、婚姻命盘子模块的成功结构，只读取已有命盘信息，不改动排盘逻辑。
- 模块基于六亲、掌诀、地支、五行、六神、五星与生克关系，输出事业趋势、财富趋势、适合行业、工作方式、优势挑战、组合建议与行动提醒。
- 底部/浮动阅读导航新增“事业”入口，方便在命运结果页快速跳转到事业子模块。
- 增加移动端宽度、换行和表格横向滚动保护，避免正文被右侧快捷按钮区域裁切。
- 同步更新页面版本记录、README、manifest 与 service-worker 缓存名。

## v10.5.22 快捷按钮内置修复版
- 修复右侧“白天/夜间”按钮仍由后续阅读模式脚本单独插入，刷新时会突然出现并造成右侧快捷区跳动的问题。
- 将“白天/夜间”正式加入 v9.9.19 统一快捷按钮系统，与“主页 / 重占 / 电脑 / 展开 / 折叠 / 置顶 / 到底”等按钮一起生成。
- 快捷按钮设置中新增“白天/夜间”，可统一开关与排序；旧快捷配置缺少该按钮时，会按默认位置插入到“折叠”和“置顶”之间。
- 修正旧阅读模式补丁的 `addToggleButton()` 检测逻辑，避免它在已有内置按钮时再额外创建一个白天/夜间按钮。
- 保留 v10.5.21 对命运婚姻子模块移动端换行、首页说明稳定和缓存更新的修复。

## v10.5.21 移动端稳定修复版
- 修复命运结果页“婚姻与伴侣参考”子模块在手机端文字被右侧区域裁切、无法完整显示的问题。
- 强化婚姻子模块的移动端宽度约束、换行、表格横向滚动和右侧安全留白。
