## v10.5.213 结果页模块顺序整理、专项建议精简与移动端阅读体验合并版

这一版继续按照合并升级节奏推进，不改起课算法、不改六宫断法，只整理结果页阅读体验：先让用户看到结论、专项重点和下一步行动，再把学习型细读放到后面。

### 本版新增

1. 结果页模块顺序整理
   - 普通问事结果页按“快速导读 → 答案摘要 → 白话总结 → 判断重心 → 问事专项 → 行动清单 → 综合判断 → 三象过程 → 行动建议 → 学习型模块”的顺序阅读；
   - 三数落宫、排盘、单宫、双宫、象意、雷达和术语解释保留，但不再抢在专项判断前面；
   - 命运结果页保留原有命盘阅读逻辑，不强行套普通问事顺序。

2. 问事专项建议精简
   - 在“问事专项解读”顶部新增“专项建议精简版”；
   - 先给“本题按什么看 / 结果落在哪里 / 下一步怎么验证”三条；
   - 原完整专项三象细读保留，并收进“展开完整专项三象细读”。

3. 移动端阅读体验增强
   - 加大结果页卡片间距和正文行高；
   - 移动端顶部操作按钮改为更稳定的触控布局；
   - 继续增加底部阅读留白，减少 Safari / iPad / iPhone 固定导航遮挡正文。

4. 新增 v10.5.213 发版体检
   - 新增 `xlr213ResultOrderAudit`；
   - 检查结果页顺序、专项精简面板、完整细读折叠、暗夜样式和移动端样式；
   - 已纳入 Debug release checklist。

5. 版本信息补齐
   - 更新页面 title、meta description、首页标题、版本记录、README、manifest、service-worker；
   - service worker 缓存名升级到 `xiaoliuren-v10.5.213`，避免旧版本缓存抢回。
