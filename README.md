# 小六壬占卜 · v10.5.220 扩展场景 JSON 知识库整合与高风险提示版

## v10.5.220 扩展场景 JSON 知识库整合与高风险提示版

这一版是在 v10.5.219 阶段封版后，根据用户补充的 JSON 知识库做的一次**场景知识库主线整合**。本版不改起课算法、不改六宫顺推规则、不改核心断卦逻辑，只把新增场景接入现有“场景 → JSON 知识库 → 专项建议精简 → 完整三象细读”的主线流程。

### 本版主要更新

1. 扩展场景 JSON 知识库接入
   - 接入 40 个扩展场景；
   - 覆盖宏观金融、社会公共、生活服务、法律物业、车辆保险、心理咨询等方向；
   - 第一批简版内容作为参考，实际接入以详细增强版为主。

2. 新增场景入口与分组
   - 新增“宏观金融与市场”分组；
   - 新增“公共事件与舆论”分组；
   - 生活服务、风险维权、学习身心类场景补入原有首页分组；
   - 场景按钮继续使用分组折叠逻辑，避免首页一次性铺开太长。

3. 统一映射与字段标签
   - 补齐新增 sceneKey 与中文场景名映射；
   - 补齐中文别名归一化，例如股市、大盘、比特币、签证、征信、防诈骗、物业、家政、心理咨询等；
   - 补齐 time_frame、trend_level、key_factor、volatility、security、pressure、repair_possibility 等字段标签，避免结果页出现英文生字段或 undefined。

4. 高风险场景提示
   - 金融 / 投资 / 楼市 / 汇率 / 数字货币类：加入非投资建议提示；
   - 法律 / 签证 / 保险 / 征信 / 纠纷类：加入非法律或持牌顾问建议提示；
   - 医疗 / 公共卫生 / 宠物医疗 / 心理咨询类：加入非专业诊疗提示；
   - 公共事件 / 灾害 / 舆论 / 赛事类：加入以官方通报和权威信息为准的提示。

5. 新增发版体检
   - 新增 `xlr220KnowledgeExtensionAudit`；
   - 检查 40 个新增场景是否都有按钮入口、JSON pack、六宫字段和高风险免责声明；
   - 已加入 Debug release checklist。

### 后续建议

这一版已经重新进入“知识库内容扩展”范围，建议先测试新增场景选择、结果页专项建议、完整三象细读和 Debug。若没有明显问题，下一版不要再加新场景，优先做一次 **v10.5.221 场景入口精简与显示整理小修版**，重点看按钮数量、分组顺序、移动端是否太长。若显示没问题，可以继续回到阶段稳定封版状态。
