# 小六壬占卜 · v10.5.201 稳定基准后问题修复与历史中心体验微调版

## v10.5.201 稳定基准后问题修复与历史中心体验微调版

这一版承接 v10.5.200 的“历史中心阶段收口与稳定基准版”，不继续大幅堆功能，而是基于稳定基准做历史中心体验微调：集中复查按钮事件兜底、移动端滚动、暗夜/阅读模式、导出联动和 Debug 误报收口。

### 本版新增 / 修复

1. 稳定基准后问题修复与体验微调面板
   - 新增“稳定基准后问题修复与体验微调”面板；
   - 显示微调项目、按钮检查、需复查项目和暗夜体检状态；
   - 支持按“全部微调 / 按钮事件 / 移动端 / 暗夜阅读 / 导出 Debug”查看检查结果。

2. 历史中心快捷微调入口
   - 新增切到精简视图；
   - 新增切到复盘视图；
   - 新增切到导出视图；
   - 新增刷新可见面板；
   - 新增跳到历史列表。

3. 移动端触控与滚动细节
   - 继续保留 iPad / Safari 弹窗滚动增强；
   - 补充按钮最小触控高度；
   - 补充 touch-action；
   - 补充历史列表底部 scroll-margin；
   - 补充弹窗 overscroll 控制。

4. Debug / 发版体检增强
   - 新增 `history-micro-tune-v201` 发版体检项；
   - 新增 `new-content-night-theme-v201` 暗夜/阅读模式体检项；
   - Debug 历史模块快照新增 `microTuneAudit`。

5. 暗夜 / 阅读模式同步适配
   - 新增微调面板、KPI 卡片、检查卡片、状态胶囊、操作工具栏和说明区域均已补暗夜样式；
   - 继续防止新增内容出现浅灰底、白块、浅紫字和低对比文字。

6. 版本源同步
   - 页面 title、meta、首页版本、版本内容更新、README、manifest、service-worker 已同步到 v10.5.201；
   - 主线 source 已统一到 `mainline-v10.5.201`；
   - service worker cache 已更新为 `xiaoliuren-v10.5.201`。

### 本版检查

- 已检查主脚本语法。
- 已检查 manifest JSON。
- 已检查重复 ID。
- 已检查旧版本 source 残留。
- 已检查 PDF 旧版本硬编码残留。
- 已检查 ZIP 完整性。
- 本版只优化历史中心体验微调、按钮兜底、移动端滚动、Debug 体检和暗夜/阅读模式，不修改核心起课算法、落宫规则、断卦逻辑和 JSON 知识库。

### 下一版建议

v10.5.202 可以做“历史中心精简视图默认化与面板折叠记忆版”：继续减少历史中心打开后的视觉压力，把常用面板默认化、折叠状态记忆化，并进一步优化手机端首次打开体验。
