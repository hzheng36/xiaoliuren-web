## v10.5.212 结果页行动清单、复盘快捷入口与白话建议合并版

这一版回到正常合并升级节奏，不再只修单点问题。重点把结果页从“看懂解释”继续推进到“知道接下来怎么做”。

### 本版新增

1. 结果页新增下一步行动清单
   - 自动整理为“现在先做 / 接下来观察 / 复盘时记录”三步；
   - 同时显示当前场景、占问事项和三象顺序；
   - 更适合普通问事结果页快速落地。

2. 新增复制行动清单
   - 可一键复制场景、问题、三象、三步行动和日常提醒；
   - 方便粘贴到备忘录、聊天记录或后续复盘里。

3. 新增保存并复盘入口
   - 当前结果未保存时，会先写入历史记录；
   - 然后直接打开复盘弹窗；
   - 不修改起课数字、落宫、断卦结论或知识库内容。

4. 白话建议继续日常化
   - 行动清单不再堆术语；
   - 更强调现实里要先做什么、观察什么、以后怎么验证。

5. 暗夜 / 阅读模式同步适配
   - 新增行动清单面板、卡片、按钮和提示已补暗夜样式；
   - 继续避免浅灰底、白块、浅紫字和低对比文字。
