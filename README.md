## v10.5.167 PDF 明细舒展与分页优化版

这一版保留 v10.5.166 已经确认好看的 PDF 报告整体视觉，重点优化用户反馈的“明细区太拥挤”问题。

### 本版新增 / 修复

1. PDF 历史明细从密集表格改为“报告卡片式明细”。
2. 每条历史记录独立展示：
   - 时间
   - 场景
   - 识别类型
   - 起课方式
   - 三数
   - 落宫
   - 末象
   - 占问事项
   - 直接答案
   - 综合判断摘要
   - 三象摘要
   - 最新复盘摘要
3. PDF 复盘明细从表格改为独立复盘卡片，展示：
   - 占卜时间
   - 复盘时间
   - 场景
   - 起课方式
   - 三数 / 落宫 / 末象
   - 准确度
   - 准确单位
   - 实际结果
   - 复盘备注
   - 更新时间
4. 长文本自动换行，不再强行挤进一格。
5. 打印分页增加 `break-inside: avoid` / `page-break-inside: avoid`，尽量避免一张明细卡片被切成上下两半。
6. PDF 报告封面、KPI、关键发现、场景排行、末象落点和图表总览维持 v10.5.166 的视觉风格。
7. Debug 快照与发版体检新增 `PDF 明细舒展卡片检查通过`。

### 使用方法

1. 打开网站。
2. 进入历史分析或历史记录。
3. 点击 `导出 PDF 报告`。
4. 浏览器会打开打印窗口。
5. 在打印选项中选择 `保存为 PDF`。
