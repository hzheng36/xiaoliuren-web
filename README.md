# 小六壬占卜 · v10.5.180 历史复盘时间线与版本源清理版

## v10.5.180 历史复盘时间线与版本源清理版

这一版承接 v10.5.179 的“标签 / 备注 / 归档”历史整理能力，进一步把历史记录串成可长期复盘的时间线；同时清理 PDF 报告封面的旧版本硬编码，减少新旧版本残留和版本源不统一风险。

### 本版新增

1. 历史复盘时间线
   - 历史记录中心新增“列表视图 / 时间线视图”切换；
   - 时间线按日期分组显示历史记录；
   - 每条记录继续显示场景、问题、起课方式、三宫落宫、标签、归档状态和复盘状态；
   - 时间线视图不修改历史数据，只改变阅读和查找方式。

2. 待复盘入口
   - 新增“待复盘”快捷按钮；
   - 自动筛出暂无复盘、未填写实际结果，或准确度仍为“待验证”的记录；
   - 方便长期使用后回头补充实际结果，形成个人案例库。

3. 复盘搜索增强
   - 历史搜索范围扩展到标签、整理备注、复盘实际结果、复盘备注和准确度状态；
   - 仍支持搜索场景、问题、起课方式、取数和落宫；
   - 与日期、场景、收藏、标签、归档筛选保持联动。

4. PDF 版本源清理
   - 修复 PDF 报告封面残留 `PDF Report · v10.5.178` 的硬编码问题；
   - PDF 封面版本号改为跟随 `XLR_SOURCE_VERSION / APP_VERSION` 自动显示；
   - 减少后续发版时旧版本文字残留和版本跳动隐患。

5. Debug / 发版体检增强
   - 新增 `history-timeline-v180` 发版体检项；
   - 检查时间线按钮、待复盘入口、时间线渲染函数、待复盘判断函数和增强搜索函数是否完整挂载；
   - 继续保留 v10.5.178 批量管理与 v10.5.179 标签/备注/归档审计。

### 本版检查

- 已同步更新页面 title、meta、首页版本标题、版本记录、README、manifest、service-worker。
- 已检查并清理 PDF 封面旧版本硬编码。
- 已检查主脚本语法。
- 已保留 v10.5.176 iPad Excel 导出兼容体检逻辑。
- 已保留 v10.5.177 历史分析图表钻取与明细联动逻辑。
- 已保留 v10.5.178 历史记录批量管理、选中导出与撤销保护。
- 已保留 v10.5.179 历史标签、备注、归档和整理筛选。
- 本版只增强历史记录阅读/复盘管理层，不修改核心起课算法、落宫规则、断卦逻辑和 JSON 知识库。

### 下一版建议

v10.5.181 可以继续做“标签统计与复盘洞察增强版”：基于标签统计数量、待复盘数量和不同标签下的复盘准确率，让历史案例库真正变成个人经验分析面板。
