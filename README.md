# 小六壬占卜 · v10.5.182 历史案例库增强版

## v10.5.182 历史案例库增强版

这一版承接 v10.5.179 的标签/备注/归档、v10.5.180 的时间线/待复盘入口，以及 v10.5.181 的标签统计与复盘洞察，把历史记录进一步升级成“可长期查阅的个人案例库”。

### 本版新增

1. 历史案例库摘要面板
   - 历史记录中心新增“历史案例库摘要”；
   - 自动统计当前筛选范围内的案例总数、高价值案例、已验证案例、待复盘案例和平均准确率；
   - 会跟随日期、场景、收藏、归档、搜索、标签、复盘状态、案例筛选和排序范围刷新。

2. 高价值案例标记
   - 单条历史记录新增“高价值 / 取消高价值”按钮；
   - 批量管理新增“标记高价值 / 取消高价值”；
   - 高价值案例只属于历史整理字段，不影响核心起课、落宫、断卦和知识库判断。

3. 案例快捷筛选与排序
   - 新增“全部案例 / 高价值案例 / 已验证案例 / 待复盘案例”快捷筛选；
   - 新增排序：最新优先、最早优先、高价值优先、已验证优先、准确率优先、待复盘优先；
   - 当前案例筛选会同步影响历史记录显示、统计面板和导出范围预览。

4. 案例摘要卡片
   - 历史记录卡片新增“案例摘要”；
   - 把三宫落宫、直接结论、复盘状态和标签串成一条快速阅读摘要；
   - 案例库面板会自动挑选当前范围内的高价值/已验证案例作为摘要卡片。

5. Debug / 发版体检增强
   - 新增 `history-case-library-v182` 发版体检项；
   - 检查案例库工具栏、案例库面板、KPI、排序选择器、高价值按钮、批量标记和审计函数是否完整挂载；
   - 继续保留 v10.5.178 批量管理、v10.5.179 标签/归档、v10.5.180 时间线/待复盘、v10.5.181 标签洞察审计。

### 本版检查

- 已同步更新页面 title、meta、首页版本标题、版本记录、README、manifest、service-worker。
- 已统一主线 source 标记为 `mainline-v10.5.182`。
- 已检查主脚本语法、重复 ID、旧版本残留和 ZIP 完整性。
