# 小六壬占卜 · v10.5.190 经验库导出与复盘质量仪表盘版

## v10.5.190 经验库导出与复盘质量仪表盘版

这一版承接 v10.5.189 的“复盘模板应用与经验库筛选”，继续把案例经验库从“能筛选、能回看”推进到“能导出、能汇总、能沉淀”。本版重点不修改核心占卜逻辑，而是把当前经验库筛选范围接入 CSV / PDF 导出，并新增复盘质量仪表盘、模板使用排行和高价值经验摘要。

### 本版新增

1. 经验库导出与复盘质量仪表盘
   - 历史记录中心新增“经验库导出与复盘质量仪表盘”面板；
   - 自动显示当前经验库筛选下的可导出经验数、平均评分、已复盘经验和应期经验；
   - 支持按当前经验筛选范围查看复盘质量分布。

2. 案例经验库 CSV / PDF 导出
   - 新增“导出经验 CSV”；
   - 新增“导出经验 PDF”；
   - 导出范围跟随当前历史筛选 + 当前经验库筛选；
   - 历史 JSON 与完整备份仍保留全量安全导出，不会被临时筛选范围误影响。

3. 复盘质量分布
   - 按案例学习评分自动分层：S级高价值、A级可参考、B级需复查、C级样本不足、待复盘；
   - 帮助快速判断当前经验库是偏高价值沉淀，还是偏复查样本。

4. 模板使用排行
   - 读取本地复盘模板使用次数；
   - 显示最常用模板和使用次数排行；
   - 帮助观察哪些模板最常用于复盘补全。

5. 高价值经验摘要
   - 自动列出当前范围内高分、高价值或代表性经验；
   - 支持一键选中高价值经验对应的历史记录；
   - 方便继续批量标记、标签、归档或导出。

6. 暗夜 / 阅读模式同步适配
   - 新增仪表盘、质量卡片、质量进度条、导出按钮、导出说明均已加入暗夜模式覆盖；
   - 继续防止新增内容出现浅灰底、白块残留、浅紫字、低对比文字、按钮/标签未继承主题变量等问题；
   - 新增内容暗夜/阅读模式检查继续纳入固定发版体检。

7. Debug / 发版体检增强
   - 新增 `history-quality-dashboard-v190` 发版体检项；
   - 新增 `new-content-night-theme-v190` 发版体检项；
   - Debug 历史模块快照新增 `qualityDashboardAudit`；
   - 经验库 CSV / PDF 导出、复盘质量分布和新增仪表盘暗夜样式均纳入发版检查。

### 本版检查

- 已同步更新页面 title、meta、首页版本标题、版本记录、README、manifest、service-worker。
- 已统一主线 source 标记为 `mainline-v10.5.190`。

- 本版只增强经验库导出、复盘质量仪表盘和暗夜/阅读体检，不修改核心起课算法、落宫规则、断卦逻辑和 JSON 知识库。
