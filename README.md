## v10.5.226 主页特性卡片修复与发版体检收口版

这一版继续基于 v10.5.225 小修收口，不改起课算法，不改核心断卦逻辑。重点修复首页特性卡片被上一版 CSS 误插入正文的问题，并把发版体检误报项收口。

### 本版主要更新

1. 首页特性卡片修复
   - 修复首页“三法起课 / 六宫排盘 / 增强解读”区域中误显示 CSS 源码的问题。
   - 将 v10.5.225 的命运清理与寻物样式重新归回主 `style` 区，不再出现在页面正文。
   - 压缩三张特性卡片文案，移动端显示更稳，不再出现内容溢出或断裂。

2. 排版与暗夜模式修复
   - 补强 `.feature-list` 和 `.feature-item` 的宽度、换行、移动端间距。
   - 同步适配暗夜模式 / Debug 夜间体检样式。

3. Debug 发版体检收口
   - 修复 v10.5.225 中 `xlr225DestinyDayHourLostAudit` 因 CSS 未进入主 style 而报 `hasCss / hasNightCss / hasMobileCss` false 的问题。
   - 修复 v10.5.218 / v10.5.219 的版本复核项仍检查旧版本号导致的 `RELEASE_CHECKLIST_REVIEW` 误报。
   - 新增 `xlr226HomeFeatureAudit`，检查首页三张特性卡片、CSS 是否泄露到正文、首页卡片样式和暗夜样式。

4. 保持上一版主功能
   - 继续保留 v10.5.225 的命运冗余内容清理。
   - 继续保留日加时断 36 组原文 + 白话解释。
   - 继续保留失物专项寻物口诀、特殊物品提示和日加时寻物组合断语。
