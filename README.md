# 小六壬占卜 · v10.5.177 历史分析图表钻取与明细联动版

## v10.5.177 历史分析图表钻取与明细联动版

这一版从“筛选联动”继续往前走，把历史分析里的图表做成可点击的钻取入口。以后不是只能看柱状图，而是可以点某一个月份、场景、宫位、末象、复盘准确度，直接看到对应的历史记录和复盘明细。

### 本版新增

1. 历史分析图表点击钻取
   - 按月份分布；
   - 按星期分布；
   - 场景分布；
   - 起课方式分布；
   - 六宫出现频率；
   - 末象落点分布；
   - 识别类型分布；
   - 复盘准确度分布；
   - 已复盘场景分布。

2. 当前图表筛选明细
   - 点击图表后显示“当前图表筛选明细”；
   - 显示命中历史记录数、复盘明细数；
   - 预览前 8 条命中记录；
   - 下方历史明细表、复盘明细表同步收窄。

3. 导出范围联动
   - XLSX 报表跟随当前筛选 + 图表钻取范围；
   - PDF 完整 / 图表 / 明细三种模式跟随当前范围；
   - 历史 TXT / 复盘 TXT / 历史 CSV / 复盘 CSV 跟随当前范围；
   - 导出确认和导出前范围预览会显示“图表：xxx”。

4. Debug 与发版体检
   - 新增 `chartDrilldownAudit`；
   - 新增 `chartDrilldownReady`；
   - 新增 `chartDrilldownFilteredCount`；
   - 新增 `analytics-chart-drilldown-v177` 发版体检项。

### 本版检查

- 已同步更新页面 title、meta、首页版本标题、版本记录、README、manifest、service-worker。
- 已检查主脚本语法。
