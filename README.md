# 小六壬占卜 · v10.5.34 主题切换性能修复版

## v10.5.34 主题切换性能修复版

- 修复白天/夜间切换时卡顿：切换期间临时禁用全页面 transition、animation 和 backdrop-filter。
- 暗夜白块兜底扫描改为避开主题切换窗口，并加入节流与数量上限，减少命运长页面全量重扫造成的 1 秒左右卡顿。
- 白天/夜间快捷按钮只更新主题 class 与按钮文字，不触发结果页重渲染、导航重建或版本标题重写。
- 保留 v10.5.33 已确认修好的命运主导航与二级导航顺序，不改排盘逻辑和事业/健康/婚姻子模块内容。
- 同步更新页面版本记录、README、manifest 与 service-worker 缓存名，并检查旧补丁/版本跳动残留。

## v10.5.33 口诀与综合导航顺序修复版

- 修复命运结果页滚到“综合判断”区域时，主导航误高亮“口诀”的问题。
- 将命运“本课用到的口诀”模块重新固定到“三象解读”之后，让页面实际 DOM 顺序与底部主导航顺序一致。
- 保留 v10.5.32 的命运二级导航主从同步逻辑：命盘 / 健康 / 婚姻 / 事业只在命盘子模块范围内显示。
- 不新增轮询，不改排盘逻辑，不改事业、健康、婚姻子模块内容，避免再次引起页面抖动。
- 同步更新页面版本记录、manifest 与 service-worker 缓存名，并重新检查旧补丁抢回旧版本问题。

## v10.5.32 命运二级导航主从同步修复版

- 继续修复命运二级导航显示范围：二级导航现在会跟主阅读导航同步。
- 当主导航当前阅读位置已经是“口诀、综合、三象、雷达”等主模块时，命盘 / 健康 / 婚姻 / 事业二级导航会强制隐藏。

- 保留 v10.5.30 / v10.5.31 的稳定方案，不新增轮询，不改排盘逻辑，不改主导航居中逻辑，降低再次抖动风险。

- 同步更新页面版本记录、README、manifest 与 service-worker 缓存名。
