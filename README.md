## v10.5.198 历史中心批量操作确认与误触保护版

这一版承接 v10.5.197 的“历史中心搜索体验与筛选记忆版”，继续集中整理历史中心的实际使用安全性。重点是降低批量删除、批量标记、批量归档、标签处理、单条删除和清空历史时的误触风险，同时继续保持新增内容暗夜/阅读模式同步适配。

### 本版新增 / 修复

1. 历史中心批量操作确认与误触保护面板
   - 新增“历史中心批量操作确认与误触保护”面板；
   - 显示保护模式、删除口令、已选记录、撤销备份状态；
   - 保护只影响历史中心确认流程，不修改核心起课、落宫、断卦或知识库逻辑。

2. 批量操作统一确认
   - 批量收藏 / 取消收藏；
   - 批量标记高价值 / 取消高价值；
   - 批量加标签 / 移除标签；
   - 批量归档 / 取消归档；
   - 批量删除、全选当前范围；
   - 都统一进入确认流程，减少移动端误触。

3. 删除类高风险保护
   - 删除选中、删除单条、清空全部历史记录加入高风险确认；
   - 开启删除口令时，需要输入“删除”或“清空”才会继续；
   - 删除类操作执行前会保存撤销备份。

4. 撤销保护增强
   - 批量修改继续保存撤销备份；
   - 单条删除现在也保存撤销备份；
   - 清空全部历史记录前也保存撤销备份；
   - 历史中心批量状态会提示当前是否存在可撤销操作。

5. 移动端误触保护
   - 面板提供“安全全选当前”和“取消当前选择”；
   - 当前选中数量、筛选范围和保护状态会明确显示；
   - 适合 iPhone / iPad 上批量整理历史记录时避免误点。

6. 暗夜 / 阅读模式同步适配
   - 新增批量保护面板、保护卡片、工具栏按钮和批量状态提示已补暗夜样式；
   - 继续防止新增内容出现浅灰底、白块、浅紫字和低对比文字。

7. Debug / 发版体检增强
   - 新增 `history-batch-safety-v198` 发版体检项；
   - Debug 历史模块快照新增 `batchSafetyAudit`；
   - `new-content-night-theme-v198` 纳入新增批量保护面板暗夜/阅读模式检查。

8. 版本源同步
   - 页面 title、meta、首页版本标题、版本记录、README、manifest、service-worker 已同步到 v10.5.198；
   - 主线 source 已统一到 `mainline-v10.5.198`；
   - service worker cache 已更新为 `xiaoliuren-v10.5.198`。
