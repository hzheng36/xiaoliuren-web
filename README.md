# 小六壬个人占卜台

当前版本：**v9.9.13 按钮修复版**

这是一个纯前端的小六壬个人网页版工具，可部署在 GitHub Pages 上使用。工具不需要服务器、不联网、不接 API，所有占卜记录、复盘、模板和设置都保存在当前浏览器本地。

## 核心起课规则

本工具采用 **三数连续顺推法** 作为唯一正式起课规则：

1. 六宫顺序：大安 → 留连 → 速喜 → 赤口 → 小吉 → 空亡。
2. 第一数：从大安起数，落到第一宫。
3. 第二数：从第一数落宫继续顺推，落到第二宫。
4. 第三数：从第二数落宫继续顺推，落到第三宫。
5. 时辰起课：取农历月、农历日、时辰序数，再按连续顺推落宫。
6. 事件起课：按事件文字生成三数，再按连续顺推落宫。

## 主要功能

- 数字起课、时辰起课、事件起课
- 场景化问事断法与白话解读
- 简洁版 / 详细版结果切换
- A/B 选择对比
- 问题模板、我的模板、模板收藏、模板批量管理
- 场景规则查询
- 历史记录、历史分析、日期筛选
- 多条复盘时间线
- TXT 导出、历史 JSON 导入导出、完整备份 JSON 导入导出
- 阅读设置、章节卡点导航、阅读进度条

## 本地保存说明

本工具的数据保存在浏览器本地 `localStorage` 中。换设备、换浏览器或清理缓存前，请先导出完整备份 JSON。

- TXT：适合阅读、分享、人工分析。
- 历史 JSON：适合迁移历史记录、收藏、复盘和 A/B 数据。
- 模板 JSON：适合迁移自定义问题模板。
- 完整备份 JSON：适合完整迁移整个个人占卜台。

## GitHub Pages 部署

1. 将 `index.html` 和 `README.md` 上传到 GitHub 仓库根目录。
2. 进入仓库 `Settings → Pages`。
3. Source 选择 `Deploy from a branch`。
4. Branch 选择 `main`，Folder 选择 `/root`。
5. 保存后等待几分钟，即可通过 GitHub Pages 地址访问。
