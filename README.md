## v10.5.227 普通问事术语雷达导航与发版警告收口版

这一版继续做可见问题收口，不改起课算法、不改三数连续顺推法、不改历史记录和复盘数据。重点修复普通问事结果页底部导航里“术语 / 图表”的顺序、命名和跳转问题，并把上一版还残留的发版体检警告收口。

### 本版主要更新

1. 普通问事导航顺序修复
   - 修复普通问事结果页中“术语解释”和“雷达分析图”实际显示顺序与底部导航顺序相反的问题。
   - 结果页主线顺序调整为：结果 → 导读 → 综合 → 三象 → 单宫 → 双宫 → 术语 → 雷达。
   - 历史回看模式也同步使用同一套顺序。

2. 术语导航点击修复
   - 修复点击“术语”后，导航 active / slot-active 容易被雷达模块抢走的问题。
   - v10.5.213 结果页排序与 v10.5.223 普通问事主线兜底排序已统一，不再互相抢顺序。

3. “图表”改名为“雷达”
   - 普通问事底部导航中的“图表”统一改为“雷达”。
   - 快速导读文案同步改为“最后看术语和雷达”。
   - Debug 导航审计中的 required nav 文案同步改为“雷达”。

4. 发版体检警告收口
   - v10.5.207 场景/占问事项联动旧面板在 v10.5.223 普通问事主线精简后已经被合并/隐藏，不再作为阻断项。
   - v10.5.209 白话总结/复盘回填旧面板同样改为“旧逻辑保留可审计，不阻断普通问事主线发布”。
   - 保留原函数与审计信息，不删除旧逻辑、不改核心断卦。

5. 新增审计
   - 新增 `xlr227OrdinaryRadarGlossaryNavAudit`。
   - 检查普通问事导航标签是否为“雷达”、导航顺序是否为“术语 → 雷达”、DOM 顺序是否为术语在雷达前。
