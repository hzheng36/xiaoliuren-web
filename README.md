# 小六壬占卜 · v10.5.21 移动端稳定修复版

基于 v10.5.20 导航与命运口诀修复版继续小步修复。

## v10.5.21 移动端稳定修复版
- 修复命运结果页“婚姻与伴侣参考”子模块在手机端文字被右侧区域裁切、无法完整显示的问题。
- 强化婚姻子模块的移动端宽度约束、换行、表格横向滚动和右侧安全留白。
- 统一首页版本说明文案，避免旧补丁刷新后改写 hero 文案造成标黄区域跳动。
- 白天/夜间快捷按钮提前参与快捷区渲染，减少刷新时右侧按钮后出现造成的抖动。
- Debug 增加 `__v10521MobileStability` 标记，可用于确认本补丁已加载。

## v10.5.20 导航与命运口诀修复版
- 修复历史结果页导航偶尔混入首页导航“顶部 / 场景 / 问事 / 起课”的问题，结果页导航现在会按当前结果完整重建。
- 命运结果页强制保留并显示“本课用到的口诀”，导航中同步出现“口诀”。
- 打开历史结果后重新同步 `result-page` / `result-destiny` 状态，减少普通模板和命运模板互相污染。
- Debug 日志增加 `overall`、`formula`、`formulaContent` 监测项，方便判断口诀模块是否存在、显示和有内容。
