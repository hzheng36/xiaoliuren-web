## v10.5.171 PDF 导出确认与明细收口版

这一版基于 v10.5.170 继续收口 PDF 报告导出层，不增加自定义标题/备注输入，重点把导出前确认、明细模式封面摘要、打印分页细节和 Debug 审计补齐。核心目标是：导出前先知道“这次会导出什么、多少条、会省略什么”，避免误导出，同时继续保留 v10.5.166 已认可的 PDF 视觉风格。

### 本版新增 / 修复

1. PDF 导出前确认摘要：
   - 点击导出完整 PDF、只导出图表 PDF、只导出明细 PDF 前，会弹出确认摘要。
   - 摘要显示导出模式、筛选范围、历史记录条数、复盘明细条数、包含内容和省略内容。
   - 用户确认后才打开打印窗口，避免误选模式或误导出全部历史。
2. 只导出明细 PDF 封面增强：
   - 明细模式封面直接显示历史记录数和复盘明细数。
   - 明确说明已省略图表和部分摘要模块，只保留逐条历史与复盘明细。
   - 正文新增“明细导出摘要”，方便保存 PDF 后核对导出范围。
3. PDF 明细分页继续收口：
   - 继续保留舒展卡片、字段分组和长文本自动换行。
   - 打印样式加强 detail-card、detail-head、detail-meta、detail-group、detail-field 的防断页策略。
   - 对长文本字段和分区增加 orphans / widows 保护，减少打印时标题和内容分离。
4. Debug / 发版体检增强：
   - 新增 `ANALYTICS_PDF_EXPORT_CONFIRM_MISSING` 警告项。
   - 新增 `PDF 导出确认与明细封面摘要检查通过` 通过项。
   - Debug 快照会记录 PDF 导出确认摘要是否生成、明细模式封面摘要是否存在、导出条数预览是否可用。
5. 继续保持核心隔离：
   - 本版只改导出报告层、PDF 预览确认和 Debug 审计。
   - 不修改起课算法、落宫规则、历史记录结构、复盘数据和 JSON 知识库断法。

### 使用方法

1. 打开网站。
2. 进入历史分析或历史记录。
3. 在导出区选择：
   - `导出完整 PDF`
   - `只导出图表 PDF`
   - `只导出明细 PDF`
4. 先查看弹出的“PDF 导出确认”摘要。
5. 确认无误后继续，浏览器会打开打印窗口。
6. 在打印选项中选择 `保存为 PDF`。
