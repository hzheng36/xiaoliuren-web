# 小六壬占卜 · v10.5.217 可见问题修复版 1

## v10.5.217 可见问题修复版 1

这一版正式进入阶段收尾后的“可见问题修复版 1”。本版不改起课算法、不改六宫断法、不新增大功能，只针对肉眼已经看到的问题做收口：Debug 诊断说明太长、历史分析阅读路径高亮不一致，以及移动端 / 暗夜模式下的可读性细节。

### 本版修复

1. Debug 诊断长说明折叠
   - Debug 弹窗顶部改为“短说明 + 展开完整说明”；
   - 默认只显示最关键的安全边界和使用提示；
   - 完整审计范围、历史版本审计说明收进可展开区域；
   - 减少 iPhone / iPad 上打开 Debug 后首屏被长文字占满的问题。

2. 历史分析高亮同步
   - 历史分析阅读路径默认高亮从“复盘明细”改为“统计摘要”；
   - 与历史记录默认高亮“跳到历史列表”的逻辑保持一致：都优先提示第一阅读入口；
   - 保留“复盘明细”按钮，但不再默认突出，避免用户误以为要从复盘明细开始看。

3. 移动端与暗夜模式细节修复
   - 补强 Debug 折叠说明在暗夜模式下的文字、边框和背景对比；
   - 调整 Debug 弹窗移动端顶部、正文、说明卡片的间距；
   - 继续减少弹窗内容在 Safari / iPad / iPhone 上显得拥挤的问题。

4. 新增 v10.5.217 发版体检
   - 新增 `xlr217VisibleFixAudit`；
   - 检查 Debug 短说明、完整说明折叠、历史分析默认高亮、暗夜样式和移动端样式；
   - 已纳入 Debug release checklist。
