## v10.5.211 历史总开关间距与移动端折叠体验修正版

这一版按截图反馈做体验修复，不继续堆新功能。重点修复历史记录弹窗里“总开关”说明和“全部展开 / 全部折叠”按钮间距太小、移动端视觉拥挤的问题，并同步处理历史分析总开关。

### 本版修复

1. 历史记录总开关间距修复
   - “总开关”说明与按钮之间改为卡片式布局；
   - iPhone / iPad 下改为单列显示，说明、全部展开、全部折叠之间留出稳定间距；
   - 增强按钮触控高度，减少误触。

2. 历史分析总开关同步修复
   - 历史分析的“全部展开 / 全部折叠”总开关同步使用同一套排版；
   - 避免说明文字和按钮贴得太近。

3. 暗夜 / 阅读模式适配
   - 总开关容器、说明文字、按钮均补暗夜模式样式；
   - 继续避免浅灰底、白块、浅紫字和低对比文字。

4. Debug 体检增强
   - 新增 `xlr211FoldSpacingAudit()`；
   - 检查历史记录/历史分析总开关容器、按钮、移动端布局和暗夜 CSS 是否挂载。
