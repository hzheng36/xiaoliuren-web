# 小六壬占卜 · v10.5.183 导出报告联动增强版

## v10.5.183 导出报告联动增强版

这一版承接 v10.5.179 的标签/备注/归档、v10.5.180 的时间线/待复盘入口、v10.5.181 的标签统计与复盘洞察，以及 v10.5.182 的历史案例库，把“案例库当前筛选范围”正式接入历史记录导出中心。

### 本版新增

1. 历史导出中心跟随案例库筛选
   - 历史记录中心的 TXT / CSV 改为导出当前可见案例范围；
   - 范围会跟随搜索、日期、场景、收藏、标签、归档、待复盘、高价值案例和案例排序；
   - 导出文件名增加 `case_scope` 标记，方便和全量导出区分。

2. 历史记录页 XLSX / PDF 联动
   - 历史记录导出中心的 Excel 报表改为 `exportHistoryScopeExcelReport()`；
   - 历史记录导出中心的 PDF 报告改为 `exportHistoryScopePdfReport()`；
   - 这两个按钮不再误用历史分析页筛选条件，而是按历史案例库当前范围生成。

3. 导出前范围预览增强
   - 预览新增高价值案例数、标签数量、待复盘数量；
   - 继续显示历史记录数、复盘明细数、收藏数、涉及场景数；
   - 导出前可以更清楚确认当前范围是否正确。

4. JSON / 完整备份保持全量安全导出
   - 历史 JSON 仍用于全量历史迁移；
   - 完整备份 JSON 仍保存历史、复盘、设置、模板等完整受保护数据；
   - 避免用户临时筛选后误以为已经完成全量备份。

5. Debug / 发版体检增强
   - 新增案例库导出联动检查；
   - 检查历史范围 XLSX/PDF 函数是否挂载；
   - 检查导出中心按钮、范围预览、版本源和主线 source 是否统一。

### 本版检查

- 已同步更新页面 title、meta、首页版本标题、版本记录、README、manifest、service-worker。
- 已统一主线 source 标记为 `mainline-v10.5.183`。
- 已检查主脚本语法、重复 ID、旧版本残留和 ZIP 完整性。
- 已保留 PDF v10.5.166 以来确认好看的视觉风格。
- 本版只增强导出报告联动层，不修改核心起课算法、落宫规则、断卦逻辑和 JSON 知识库。

### 下一版建议

v10.5.184 可以做“复盘编辑与案例学习增强版”：把已有复盘记录做成可编辑、可追踪、可按准确/不准原因归类的长期学习系统。
