# 小六壬占卜 · v10.5.222 普通问事简明结果回归版

## v10.5.222 普通问事简明结果回归版

这一版是在 v10.5.221 之后做的结果页收口修复。用户反馈普通问事不想要“行动清单”等过重内容，希望回到类似 v10.5.76 的简单明了阅读方式。本版不改起课算法、不改核心断卦逻辑，只调整普通问事结果页的默认显示。

### 本版主要更新

1. 普通问事回到简明结果
   - 默认启用简洁版；
   - 优先保留答案摘要、问事专项精简、综合判断、三数落宫等核心结果；
   - 减少顶部多层导引和额外操作面板。

2. 隐藏过重的行动与导引模块
   - 普通问事默认隐藏 v10.5.212 的“下一步行动清单”；
   - 默认隐藏 v10.5.214 的“重点阅读路径”；
   - 默认清理 v10.5.221 的“先看结论 / 答案优先”面板。

3. 保留完整内容入口
   - 详细版、学习型内容、排盘、术语等未删除；
   - 用户需要深入复盘时仍可切回详细阅读。

4. 暗夜模式与移动端适配
   - 新增简明结果状态样式；
   - 手机端保持核心结果卡片间距；
   - 避免隐藏模块留下空白或遮挡。

5. 新增发版体检
   - 新增 `xlr222SimpleResultAudit`；
   - 检查简明结果模式、行动清单隐藏、答案优先面板清理、暗夜样式和移动端样式；
   - 已加入 Debug release checklist。

### 后续建议

这一版建议重点测试普通问事：起课后是否一眼能看到答案摘要、专项精简和综合判断；是否不再出现“下一步行动清单”和“先看结论”面板。若仍觉得内容多，下一版可以继续做“结果页只保留核心四块”的更极简模式。
