## v10.5.204 普通问事结果页体验、场景知识库与默认折叠合并增强版

这一版开始按“合并升级节奏”推进，不再只围绕单点小功能开版本。本版一次合并普通问事结果页体验、场景知识库提示、设置/历史记录/历史分析默认折叠、暗夜/阅读模式和 Debug 版本源检查。

### 本版新增 / 修复

1. 普通问事结果页体验增强
   - 普通问事结果页新增“快速读法”提示；
   - 引导用户按“答案摘要 → 三象过程 → 问事专项 → 行动建议”的顺序阅读；
   - 只优化阅读提示，不修改起课算法、落宫规则、断卦结果和历史数据。

2. 场景知识库提示优化
   - 问事专项解读区新增场景知识库提示；
   - 明确当前场景会优先按本地 JSON 专项知识库与三象过程解释；
   - 继续保留专业免责声明和现实证据优先原则。

3. 设置页默认折叠
   - 设置页中的设置卡片默认收起；
   - 用户点击标题或展开按钮后再查看详细选项；
   - 保存、关闭、恢复默认等操作保持可见和可用。

4. 历史记录默认折叠
   - 历史记录中心的高级统计、案例库、复盘、导出、体检类面板默认收起；
   - 历史列表和基础筛选保留可见，减少首次打开历史中心的视觉压力；
   - 不删除任何数据，不影响历史筛选、导出、复盘和批量管理。

5. 历史分析默认折叠
   - 历史分析中的图表、标签洞察、明细表、复盘明细和导出中心默认收起；
   - 展开后仍可查看完整分析内容；
   - 导出范围、报表预览和筛选逻辑不变。

6. 暗夜 / 阅读模式同步适配
   - 新增普通问事提示条、场景知识库提示、折叠按钮、设置折叠卡片和分析折叠区均已补暗夜样式；
   - 继续防止浅灰底、白块、浅紫字和低对比文字。

7. Debug / 发版体检增强
   - 新增 `xlr204DefaultFoldAudit()` 默认折叠体检接口；
   - 可检查设置卡片折叠、历史详情折叠、分析详情折叠、普通问事提示和暗夜 CSS 是否挂载；
   - 主线 source 统一到 `mainline-v10.5.204`。
