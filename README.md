# 小六壬占卜 · v10.5.221 普通问事答案优先与结果页减负版

## v10.5.221 普通问事答案优先与结果页减负版

这一版是在 v10.5.220 扩展场景知识库之后做的结果页阅读减负小修。新增场景后，普通问事结果页的信息量明显变大，本版不删内容、不改起课算法、不改核心断卦逻辑，只把普通问事默认阅读方式改成“先看结论，再看细节”。

### 本版主要更新

1. 普通问事新增“一屏结论”
   - 结果页顶部新增“先看结论”面板；
   - 默认显示直接结果、关键原因、下一步先做什么；
   - 用户不需要再从长篇模块里找最终判断。

2. 新增答案优先模式
   - 普通问事默认收起快速导读、白话长文、判断重心、完整专项、综合判断、三象过程、排盘和学习型模块；
   - 行动清单保留在前面；
   - 点“显示完整内容”即可恢复完整细读。

3. 快捷跳转优化
   - 新增“查看专项 / 查看行动 / 查看三象”按钮；
   - 点击后自动展开完整内容并跳到对应模块；
   - 适合手机端快速定位。

4. 暗夜模式与移动端适配
   - 一屏结论面板已适配暗夜模式；
   - 手机端卡片改为单列；
   - 按钮在小屏下保持足够触控高度。

5. 新增发版体检
   - 新增 `xlr221AnswerFirstAudit`；
   - 检查一屏结论面板、默认减负隐藏规则、展开按钮、暗夜样式和移动端样式；
   - 已加入 Debug release checklist。

### 后续建议

这一版建议重点测试普通问事结果页：起课后是否能第一眼看到结论、点“显示完整内容”是否能恢复所有模块、手机端是否比 v10.5.220 更容易阅读。若没有明显问题，下一版可以只做小范围可见问题修复，不再新增大功能。
