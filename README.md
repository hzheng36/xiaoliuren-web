## v10.5.166 PDF 图表报告基础版

这一版开始加入历史分析的 PDF 图表报告能力。

### 本版新增

1. 历史分析弹窗新增 `导出 PDF 报告`。
2. 历史记录弹窗也新增 `导出 PDF 报告`。
3. PDF 报告使用浏览器打印/保存为 PDF：
   - 不联网
   - 不上传历史数据
   - 不引入外部 PDF 库
   - 不修改历史记录、复盘、模板或占卜核心逻辑
4. PDF 报告包含：
   - 封面摘要
   - KPI 指标
   - 关键发现
   - 复盘质量
   - 场景排行
   - 末象落点
   - 图表总览
   - 历史明细预览
   - 复盘明细预览
5. 图表总览覆盖：
   - 月份分布
   - 星期分布
   - 场景分布
   - 起课方式分布
   - 六宫出现频率
   - 末象落点分布
   - 识别类型分布
   - 复盘准确度分布
   - 已复盘场景分布
   - 按场景准确率

### 使用方法

1. 打开网站。
2. 进入历史分析。
3. 点击 `分析导出`。
4. 点击 `导出 PDF 报告`。
5. 浏览器会打开打印窗口。
6. 在打印选项里选择 `保存为 PDF`。
