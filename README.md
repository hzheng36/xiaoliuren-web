# 小六壬占卜 · v10.5.192 经验库导出字段预设与报告索引版

## v10.5.192 经验库导出字段预设与报告索引版

这一版承接 v10.5.191 的“经验库打印版式与导出明细”，继续增强案例经验库的正式报告能力。重点不是修小问题，也不修改核心起课逻辑，而是让经验库导出更像一份可长期归档、可按主题阅读、可用于复盘学习的报告。

### 本版新增

1. 经验库导出字段预设与报告索引面板
   - 历史记录中心新增“经验库导出字段预设与报告索引”面板；
   - 自动显示当前可打印经验数、字段预设、分章方式和目录章节数；
   - 与当前历史筛选、经验库筛选和导出字段模式联动。

2. 字段预设记忆
   - 新增三种报告预设：
     - 快速归档：简洁字段 + 按场景分章；
     - 复盘学习：学习重点字段 + 按标签分章；
     - 完整报告：完整明细字段 + 按场景分章；
   - 预设会保存到当前浏览器本地；
   - 只影响经验 CSV / PDF 导出，不影响历史 JSON 和完整备份。

3. PDF 报告目录索引
   - 经验库 PDF 封面后新增“报告目录 / Index”；
   - 目录自动列出每一章标题和条目数量；
   - 页码仍以浏览器打印预览为准，避免跨浏览器页码计算误差。

4. 按场景 / 标签分章
   - 新增三种分章方式：
     - 不分章；
     - 按场景分章；
     - 按学习标签分章；
   - 正文会按章节输出经验卡片；
   - 继续保留分页防断、长文本舒展和移动端打印兼容。

5. 暗夜 / 阅读模式同步适配
   - 新增报告索引面板、预设按钮、分章按钮、目录说明、索引卡片均已补暗夜样式；
   - 继续防止新增内容出现浅灰底、白块残留、浅紫字、低对比文字、按钮/标签未继承主题变量等问题；
   - 新增内容暗夜/阅读模式检查继续纳入固定发版体检。

6. Debug / 发版体检增强
   - 新增 `history-report-index-v192` 发版体检项；
   - 新增 `new-content-night-theme-v192` 发版体检项；
   - Debug 历史模块快照新增 `reportIndexAudit`；
   - 字段预设、分章方式、目录索引和新增报告索引面板暗夜样式均纳入发版检查。

### 本版检查

- 已同步更新页面 title、meta、首页版本标题、版本记录、README、manifest、service-worker。
- 已统一主线 source 标记为 `mainline-v10.5.192`。
- 已检查主脚本语法、重复 ID、旧版本残留和 ZIP 完整性。
- 本版只增强经验库导出预设、PDF 目录索引和分章报告，不修改核心起课算法、落宫规则、断卦逻辑和 JSON 知识库。
