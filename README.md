# 小六壬占卜 · v10.5.223 普通问事主线结果结构回归版

## v10.5.223 普通问事主线结果结构回归版

这一版按用户反馈，把普通问事结果页从“答案摘要、生活白话、行动清单、专项精简”等后期叠加内容中收回来，回到补丁回源码前更直接的主线阅读结构。本版不改起课算法、不改核心断卦逻辑，只调整普通问事结果页默认显示。

### 本版主要更新

1. 普通问事只保留主线结果
   - 快速导读
   - 综合判断
   - 三象解读
   - 单宫断事
   - 双宫断事
   - 术语解释
   - 雷达分析图

2. 移除冗余普通问事内容
   - 默认隐藏生活白话版、答案摘要、本场景判断重心、问事专项解读、专项精简、行动清单、答案优先、重点阅读路径、三数落宫、六宫排盘、象意补充等模块。
   - 结果页不再用多层导引和“下一步行动”抢占阅读空间。

3. 导航同步收口
   - 普通问事底部章节导航同步为：结果、导读、综合、三象、单宫、双宫、术语、图表。
   - 不再出现答案、白话、专项、行动、建议等导航入口。

4. 补丁清理
   - 清理 v10.5.207 / v10.5.209 / v10.5.212 / v10.5.221 / v10.5.222 对普通问事结果页造成的冗余显示。
   - 保留命运页和历史页逻辑，不套用普通问事主线隐藏规则。

5. 新增发版体检
   - 新增 `xlr223OrdinaryMainlineAudit`；
   - 检查普通问事主线模块、旧面板清理、暗夜样式、移动端样式。
