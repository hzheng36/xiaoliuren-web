## v10.5.214 结果页重点阅读路径与章节导航同步版

这一版继续按收尾节奏推进，不改起课算法、不改六宫断法，主要把 v10.5.213 已整理好的结果页阅读顺序同步到“重点阅读路径”和底部章节导航里，避免内容顺序、快捷按钮和导航顺序不一致。

### 本版新增

1. 结果页章节导航同步
   - 普通问事底部章节导航按“导读 → 答案 → 问事 → 白话 → 重心 → 专项 → 行动 → 综合 → 三象 → 建议 → 学习型模块”的顺序排列；
   - 补齐“白话总结”和“行动清单”导航入口；
   - 重点阅读模式下被隐藏的学习型模块不会强行出现在导航里。

2. 重点阅读路径主线化
   - 原“结果页模块精简”升级为“重点阅读路径”；
   - 顶部快捷按钮顺序统一为：看导读、先看答案、白话总结、判断重心、专项解读、行动清单、综合判断、三象过程、行动建议；
   - “切到重点阅读 / 切到完整细读”继续保留，方便在简读和细读之间切换。

3. 移动端阅读按钮优化
   - 重点阅读路径按钮在移动端使用两列触控布局；
   - 行动清单入口增加轻微强调，方便用户从结论直接进入落地步骤；
   - 继续保留 v10.5.213 的结果页间距、行高和底部阅读留白。

4. 新增 v10.5.214 发版体检
   - 新增 `xlr214ReadingNavAudit`；
   - 检查底部章节导航顺序、重点阅读按钮顺序、行动清单入口、白话总结入口、暗夜样式和移动端样式；
   - 已纳入 Debug release checklist。

5. 版本信息补齐
   - 更新页面 title、meta description、首页标题、版本记录、README、manifest、service-worker；
   - service worker 缓存名升级到 `xiaoliuren-v10.5.214`，避免旧版本缓存抢回。
