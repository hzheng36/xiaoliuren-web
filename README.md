## v10.5.187 复盘学习闭环与暗夜体检强化版

这一版承接 v10.5.186 的“案例库快捷复查与新模块暗夜主题修复”，继续把历史记录中心推进成长期复盘学习工具。本版重点不是改核心占卜逻辑，而是把低分、应期偏差、未归类、待补学习要点和待复盘案例整理成更清楚的“下一步处理清单”，同时把新增内容的暗夜/阅读模式体检固定到发版流程里。

### 本版新增

1. 新增“复盘学习闭环与暗夜体检”面板
   - 自动统计当前历史筛选范围内的闭环处理事项；
   - 覆盖低分案例、应期偏差、未填原因归类、待补学习要点、待复盘记录；
   - 每类事项给出下一步处理建议，方便长期沉淀案例库经验。

2. 复盘闭环快捷处理
   - 支持“全部闭环 / 低分优先 / 应期复查 / 补原因 / 补学习 / 补复盘”切换；
   - 支持一键定位到快捷复查清单；
   - 支持联动历史搜索；
   - 支持批量选中当前闭环范围，继续使用批量收藏、标签、归档等现有管理能力；
   - 支持把当前闭环范围标记为高价值案例。

3. 暗夜/阅读模式体检强化
   - 新增暗夜主题覆盖、阅读模式继承、发版体检规则三张体检卡；
   - 继续检查新模块是否出现浅灰底、白块残留、浅紫字、低对比文字、按钮/标签未继承主题变量等问题；
   - 后续新增任何模块、卡片、弹窗、统计面板、导出预览，都必须同步检查暗夜模式与阅读模式。

4. Debug / 发版体检增强
   - 新增 `history-review-loop-v187` 发版体检项；
   - 新增 `new-content-night-theme-v187` 发版体检项；
   - Debug 历史模块快照新增 `reviewLoopAudit`；
   - 新增内容暗夜/阅读模式检查继续纳入固定发版体检。

### 本版检查

- 已同步更新页面 title、meta、首页版本标题、版本记录、README、manifest、service-worker。
- 已统一主线 source 标记为 `mainline-v10.5.187`。
- 已检查主脚本语法、重复 ID、旧版本残留和 ZIP 完整性。
- 本版只增强复盘学习闭环、批量整理入口和暗夜/阅读体检，不修改核心起课算法、落宫规则、断卦逻辑和 JSON 知识库。

v10.5.188 可以做“复盘建议模板与案例经验沉淀版”：根据不同偏差原因自动给出复盘填写提示，把常见偏差沉淀为可复用的案例经验模板，同时继续保持新增内容暗夜/阅读模式同步适配。
