# 小六壬占卜 · v10.5.175 历史分析筛选联动增强版

## v10.5.175 历史分析筛选联动增强版

这一版不再继续围绕 PDF 明细误报打转，而是回到正常功能升级：把历史分析从单一日期筛选，升级为多维筛选联动分析。

### 本版新增 / 修复

1. 历史分析新增筛选条件：
   - 日期筛选继续保留；
   - 新增场景筛选；
   - 新增起课方式筛选；
   - 新增收藏状态筛选；
   - 新增复盘准确度筛选：未复盘、已复盘、已验证、准确、部分准确、不准确、待验证。

2. 筛选结果统一联动：
   - 页面 KPI 跟随筛选结果变化；
   - 图表跟随筛选结果变化；
   - 历史明细表跟随筛选结果变化；
   - 复盘明细表跟随筛选结果变化；
   - 报表预览跟随筛选结果变化；
   - XLSX 图表报表跟随筛选结果导出；
   - PDF 完整 / 图表 / 明细三种模式跟随筛选结果导出；
   - 导出前范围预览显示当前筛选范围和条数。

3. 新增一键重置筛选：
   - 可一次清空日期、场景、起课方式、收藏状态和复盘准确度筛选。

4. 导出中心增强：
   - 历史分析导出中心的 TXT / CSV 改为“筛选历史 / 筛选复盘”导出；
   - XLSX / PDF / 筛选 TXT / 筛选 CSV 使用同一批当前分析范围；
   - 导出前范围预览会显示筛选后的历史记录数、复盘明细数、收藏数和场景数。

5. Debug 与发版体检增强：
   - 新增 `analytics-filter-linkage-v175` 发版体检项；
   - 新增筛选联动审计，检查筛选控件、主函数、筛选记录数和 `metrics.total` 是否一致；
   - 保持 1 个主 style + 1 个主 script 的源码结构。

### 本版检查
