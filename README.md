# 小六壬个人占卜台 · v9.9.25 历史查看与删除确认修正版

这是一个纯前端的小六壬网页工具，可部署到 GitHub Pages。数据保存在当前浏览器本地，支持 TXT / JSON 导出导入与 PWA 离线缓存。

## 起课规则

本工具正式主规则为 **三数连续顺推法**：

1. 第一数从大安起。
2. 第二数从第一数落宫继续起。
3. 第三数从第二数落宫继续起。

时辰起课以用户设备显示的当地时间为准；数字起课与事件起课不依赖地理位置。

## 起课注意事项

- 问题要清晰、具体。
- 一事不二占，一时不起二课。
- 问事时心念专一，不疑不妄。

## 命运场景说明

命运场景以出生时间起象，仅作传统象意、自我观察与人生规划参考，不作为人生定论。

## 数据说明

- TXT：适合阅读、分享、人工复盘。
- 历史 JSON：只迁移历史记录、收藏、复盘、A/B 数据。
- 模板 JSON：只迁移问题模板。
- 完整备份 JSON：迁移整个个人占卜台，包括历史、复盘、模板、设置等。
