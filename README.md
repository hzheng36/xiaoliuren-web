# 小六壬占卜 · v10.5.188 复盘建议模板与案例经验沉淀版

## v10.5.188 复盘建议模板与案例经验沉淀版

这一版承接 v10.5.187 的“复盘学习闭环与暗夜体检强化”，继续把历史记录中心推进成长期案例学习工具。本版重点不是改核心占卜逻辑，而是根据不同复盘状态和偏差原因，自动生成可复制的复盘填写提示，把待复盘、低分、应期偏差、未归类、待补学习和高分案例沉淀成更容易复用的经验模板。

### 本版新增

1. 新增“复盘建议模板与案例经验沉淀”面板
   - 自动统计当前历史筛选范围；
   - 按待复盘、低分复查、应期偏差、补原因、补学习、高分沉淀生成模板卡片；
   - 每张模板卡片显示命中样本数量、平均评分、复盘填写提示和经验沉淀说明。

2. 模板卡片快捷操作
   - 支持复制复盘建议模板；
   - 支持定位到快捷复查清单；
   - 支持联动历史搜索；
   - 支持批量选中当前模板对应的案例范围；
   - 支持把对应范围标记为高价值案例。

3. 案例经验沉淀增强
   - 低分案例用于沉淀“偏差类型和边界”；
   - 应期偏差案例用于沉淀“时间感和过程判断”；
   - 高分案例用于沉淀“可复用判断路径”；
   - 学习要点鼓励写成短句，方便长期搜索和回看。

4. 暗夜 / 阅读模式继续固定检查
   - 新增模板面板、模板卡片、模板工具栏、模板说明区域均已加入暗夜模式覆盖；
   - 继续防止新增内容出现浅灰底、白块残留、浅紫字、低对比文字、按钮/标签未继承主题变量等问题；
   - 新增内容暗夜/阅读模式检查继续纳入固定发版体检。

5. Debug / 发版体检增强
   - 新增 `history-review-template-v188` 发版体检项；
   - 新增 `new-content-night-theme-v188` 发版体检项；
   - Debug 历史模块快照新增 `reviewTemplateAudit`；
   - 模板面板同时纳入新增内容暗夜/阅读模式审计。

### 本版检查

- 已同步更新页面 title、meta、首页版本标题、版本记录、README、manifest、service-worker。
- 已统一主线 source 标记为 `mainline-v10.5.188`。

- 本版只增强复盘模板、案例经验沉淀和暗夜/阅读体检，不修改核心起课算法、落宫规则、断卦逻辑和 JSON 知识库。
