## v10.5.168 PDF 导出选项增强版

这一版基于 v10.5.167 继续升级 PDF 报告导出层：保留 v10.5.166/v10.5.167 已确认好看的 PDF 视觉风格和舒展明细卡片，不重做整体版式，重点新增 PDF 导出选项。

### 本版新增 / 修复

1. PDF 导出新增三种模式：
   - 完整报告：封面摘要 + KPI + 关键发现 + 图表总览 + 历史明细 + 复盘明细。
   - 只导出图表：封面摘要 + 关键发现 + 场景/末象概览 + 图表总览。
   - 只导出明细：封面摘要 + 历史明细 + 复盘明细。
2. 历史分析弹窗和历史记录弹窗都新增三种 PDF 按钮，方便从任一入口导出。
3. PDF 封面补强导出元信息：
   - 导出范围
   - 筛选说明
   - 导出模式
   - 封面日期
   - 页码打印提示
4. 继续保留 PDF 历史明细/复盘明细的舒展卡片布局，避免长文本挤在表格里。
5. Debug 快照新增完整/图表/明细三种 PDF 模式检查。
6. 发版体检新增 `PDF 导出选项检查通过`。

### 使用方法

1. 打开网站。
2. 进入历史分析或历史记录。
3. 在导出区选择：
   - `导出完整 PDF`
   - `只导出图表 PDF`
   - `只导出明细 PDF`
4. 浏览器会打开打印窗口。
5. 在打印选项中选择 `保存为 PDF`。
