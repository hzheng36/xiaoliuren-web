## v10.5.205 结果页模块精简、命运页阅读体验与场景规则整理合并版

这一版继续按“合并升级节奏”推进，把几个相关体验更新合并到同一版完成：普通问事结果页模块精简、命运页阅读体验、场景规则弹窗整理、移动端导航体验，以及新增内容的暗夜/阅读模式检查。

### 本版新增 / 修复

1. 结果页模块精简
   - 普通问事结果页新增“重点阅读 / 完整细读”切换；
   - 重点阅读模式会临时收起排盘、单宫、双宫、象意和雷达等学习型模块；
   - 保留答案摘要、三象过程、问事专项、综合判断、行动建议和术语解释；
   - 只改变阅读展示，不修改起课算法、落宫规则、断卦结果和历史数据。

2. 命运页阅读体验
   - 命运六亲参考盘新增“命运页阅读顺序”提示；
   - 引导用户按“起盘说明 → 六宫关系 → 自身与总览 → 主题模块 → 长期倾向”的顺序阅读；
   - 提供起盘说明、六宫关系、命盘总览、长期倾向快捷跳转。

3. 场景规则弹窗整理
   - 场景规则弹窗新增快捷分组入口；
   - 支持按全部、感情、财运、事业、健康、出行、失物等主题快速筛选；
   - 保留原搜索框和原规则库，不改变规则判断逻辑。

4. 移动端导航体验
   - 优化底部章节导航在手机 / iPad 上的触控高度和横向滑动体验；
   - 普通问事结果页新增移动端导航提示；
   - 减少底部导航按钮过密、点按困难和找不到模块的问题。

5. 默认折叠逻辑继续保留
   - 继续保留 v10.5.204 的设置、历史记录、历史分析默认折叠；
   - 高级内容仍可按需展开，不删除任何历史、复盘、导出和设置数据。

6. 暗夜 / 阅读模式同步适配
   - 本版新增结果页精简提示、命运页阅读提示、规则分组面板和移动端导航提示均已补暗夜样式；
   - 继续防止浅灰底、白块、浅紫字和低对比文字。

7. Debug / 发版体检增强
   - 新增 `xlr205MergedAudit()` 合并升级体检接口；
   - 新增 result-compact-v205、destiny-reading-guide-v205、scene-rules-organizer-v205、mobile-nav-experience-v205 与 new-content-night-theme-v205 检查项；
   - 主线 source 统一到 `mainline-v10.5.205`。
