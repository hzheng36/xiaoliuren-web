# 小六壬占卜 · v10.5.216 设置页、工具按钮与阅读模式最终整理版

## v10.5.216 设置页、工具按钮与阅读模式最终整理版

这一版继续按阶段收尾节奏推进，不改起课算法、不改六宫断法，重点整理设置页、阅读模式、快捷按钮和电脑/自适应模式。目标是让用户在封版前更容易看懂设置项、更容易确认当前阅读状态，也减少移动端切换模式后的混乱。

### 本版新增

1. 设置页收尾导览
   - 设置页顶部新增“设置收尾导览”；
   - 提供“日常阅读 / 夜间护眼 / 移动精简 / 电脑模式”四个预设入口；
   - 预设只先套用到设置表单，用户确认后再点“保存设置”，避免误改。

2. 阅读模式预览
   - 阅读模式区新增“阅读预览”；
   - 实时显示当前主题、背景、字号和夜间快捷按钮状态；
   - 便于确认结果页、历史记录、复盘明细等长文本区域是否适合阅读。

3. 客户端模式校准
   - 界面显示区新增“当前客户端模式”；
   - 可查看是否存在手动电脑/自适应模式记忆；
   - 新增“清除手动模式记忆”按钮，避免之前切换电脑模式后下次打开继续沿用旧状态。

4. 快捷按钮与移动端体验收口
   - 统一浮动快捷按钮在自适应模式和电脑模式下的最小宽度；
   - 设置页保存按钮在有未保存变动时增加轻提示；
   - 移动端设置页按钮改成更稳定的单列/双列触控布局；
   - 新增内容同步适配暗夜模式和阅读模式。

5. 新增 v10.5.216 发版体检
   - 新增 `xlr216SettingsToolsAudit`；
   - 检查设置导览、阅读预览、客户端模式校准、快捷按钮样式、暗夜样式和移动端样式；
   - 已纳入 Debug release checklist。
