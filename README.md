# 小六壬占卜 · v10.5.197 历史中心搜索体验与筛选记忆版

## v10.5.197 历史中心搜索体验与筛选记忆版

这一版承接 v10.5.196 的“历史中心总整理与性能轻量化版”，继续优化历史记录中心的实际使用体验。重点是让搜索、筛选、标签、归档、复盘状态、案例筛选和移动端浏览更顺手，同时继续保持新增内容暗夜/阅读模式同步适配。

### 本版新增 / 修复

1. 历史中心搜索体验与筛选记忆面板
   - 新增“历史中心搜索体验与筛选记忆”面板；
   - 显示当前匹配数量、活跃筛选数量、筛选记忆状态和移动端筛选栏状态；
   - 筛选记忆只保存在当前浏览器本地，不修改历史记录、复盘内容或核心占卜逻辑。

2. 常用筛选快捷入口
   - 新增全部记录、只看收藏、待复盘、高价值、已验证、未归档、只看归档和清空筛选；
   - 快捷入口会联动历史列表、时间线、案例库、导出范围和统计面板；
   - 仅改变历史中心当前显示范围，不影响起课、落宫、断卦和知识库。

3. 筛选条件本地记忆
   - 支持保存和恢复搜索关键词、场景、日期、标签、归档状态、收藏状态、复盘状态、案例筛选、案例排序和面板视图；
   - 打开历史记录中心时可自动恢复上次筛选；
   - 新增保存当前筛选、恢复上次筛选、清空所有筛选按钮。

4. 活跃筛选摘要
   - 自动生成当前筛选条件摘要；
   - 每个筛选条件可单独移除；
   - 方便排查“为什么当前历史列表只显示这些记录”。

5. 移动端筛选栏收合
   - 新增“收合筛选栏 / 展开筛选栏”；
   - 手机和 iPad 上可临时隐藏搜索框、日期、模式、案例筛选和标签归档栏，把空间留给历史列表；
   - 继续保留 v10.5.195 / v10.5.196 的移动端滚动与长列表轻量化能力。

6. 暗夜 / 阅读模式同步适配
   - 新增搜索记忆面板、快捷筛选按钮、筛选摘要标签、收合状态都已补暗夜样式；
   - 继续防止新增内容出现浅灰底、白块、浅紫字和低对比文字。

7. Debug / 发版体检增强
   - 新增 `history-search-memory-v197` 发版体检项；
   - Debug 历史模块快照新增 `searchMemoryAudit`；
   - `new-content-night-theme-v197` 纳入新增搜索记忆面板暗夜/阅读模式检查。

8. 版本源同步
   - 页面 title、meta、首页版本标题、版本记录、README、manifest、service-worker 已同步到 v10.5.197；
   - 主线 source 已统一到 `mainline-v10.5.197`；
   - service worker cache 已更新为 `xiaoliuren-v10.5.197`。
