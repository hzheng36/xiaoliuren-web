## v10.5.215 历史记录与历史分析阅读体验整理版

这一版进入阶段收尾节奏，不改起课算法、不改六宫断法，重点整理历史记录、历史分析、复盘明细、历史回看结果页和移动端阅读体验，让后续封版前的可见问题更集中、更容易排查。

### 本版新增

1. 历史记录阅读快捷路径
   - 历史记录弹窗新增“历史阅读快捷路径”；
   - 可快速跳到历史列表、复盘闭环、导出中心；
   - 支持一键切回精简视图或收起高级面板，减少手机/iPad 上的滚动压力。

2. 历史分析阅读路径整理
   - 历史分析弹窗新增“历史分析阅读路径”；
   - 阅读顺序固定为：统计摘要 → 图表区 → 历史明细 → 复盘明细 → 导出中心；
   - 复盘明细继续优先使用移动端卡片阅读，减少表格横向挤压。

3. 历史回看结果页提示
   - 从历史记录点击“查看”后，结果页顶部新增“历史回看模式”提示；
   - 显示场景、时间、落宫、复盘条数和原占问事项；
   - 提供“返回历史记录”和“补一条复盘”快捷入口。

4. 移动端与暗夜模式收口
   - 加大历史记录卡片、按钮、复盘明细和弹窗底部安全留白；
   - 历史记录 / 历史分析新增内容已同步适配暗夜模式；
   - 避免历史弹窗、分析弹窗和复盘弹窗底部内容被固定导航遮挡。

5. 新增 v10.5.215 发版体检
   - 新增 `xlr215HistoryReadingAudit`；
   - 检查历史阅读路径、历史分析阅读路径、历史回看提示、复盘明细卡片、暗夜样式和移动端样式；
   - 已纳入 Debug release checklist。
