# 小六壬占卜 · v10.0 命盘排法校正版

这是一个纯前端的小六壬个人占卜台，可直接部署到 GitHub Pages 使用。数据保存在当前浏览器本地，不上传服务器。

## 当前版本重点

v10.0 重点校正「命运」场景的命盘排法：

- 三数落宫在命运场景中明确标注为：月宫、日宫、时宫。
- 命盘以时宫为自身，从时宫开始排地支。
- 六神按地支装：寅卯青龙，巳午朱雀，丑辰勾陈，未戌腾蛇，申酉白虎，亥子玄武。
- 五星从日落宫起，顺排：木星、火星、土星、金星、水星、天空。
- 六亲排法支持二选一：
  - 默认：五行生克法，即生我父母、克我官鬼、我生子孙、我克妻财、同我兄弟。
  - 可选：固定宫位法，即留连自身速喜子、赤口兄弟小吉官、空亡父母大安财。
- 命盘分析按「掌诀 + 六亲 + 六神 + 五星」综合解读。
- 六宫排盘增加序号，方便查看宫位顺序。

## 主要功能

- 数字起课、单数起课、时辰起课、事件起课。
- 命运场景仅使用时辰起课，适合出生时间参考盘。
- 场景化解读：感情、事业、财运、考试、失物、健康、出行、合作、消息等待、天气、求职、家宅、选择、寻人、贵人、运势、命运等。
- 简洁版 / 详细版阅读。
- 历史记录、复盘时间线、历史分析。
- TXT / JSON 导入导出与完整备份。
- 问题模板、自定义模板、模板收藏与管理。
- 阅读设置、快捷按钮自定义、章节导航、阅读进度条。

## 起课规则

本工具的主线规则为三数连续顺推法：

1. 第一数从大安起。
2. 第二数从第一数落宫继续起。
3. 第三数从第二数落宫继续起。

命运场景采用：

> 大安起月，月上起日，日上起时。

并以时宫作为自身宫进行命盘排法。
