# 小六壬占卜 · v10.5.191 经验库打印版式与导出明细增强版

## v10.5.191 经验库打印版式与导出明细增强版

这一版承接 v10.5.190 的“经验库导出与复盘质量仪表盘”，继续优化经验库导出的实际可读性。重点不是新增复杂占法，也不修改核心起课逻辑，而是让案例经验库的 CSV / PDF 更适合长期归档、打印、手机阅读和复盘学习。

### 本版新增

1. 经验库打印版式与导出明细面板
   - 历史记录中心新增“经验库打印版式与导出明细”面板；
   - 自动显示当前经验数、当前字段模式、长文本卡片数和分页防断状态；
   - 和当前历史筛选、经验库筛选联动。

2. 导出字段选择
   - 新增三种经验导出字段模式：
     - 简洁字段：适合快速归档和轻量查看；
     - 学习重点：突出原因归类、学习标签、学习要点和评分；
     - 完整明细：保留更完整的案例问题、综合判断、复盘备注和更新时间；
   - CSV / PDF 都会跟随当前字段模式导出；
   - JSON 与完整备份仍保留全量数据，不受字段模式影响。

3. 经验库 PDF 打印优化
   - 增强经验卡片分页防断；
   - 增强字段组、标题、长文本的防断页策略；
   - 长文本自动换行并保留舒展阅读版式；
   - 封面后进入正文时更适合打印与保存 PDF。

4. 移动端 / Safari 打印兼容
   - 增加移动端单列明细样式；
   - 优化 iPad / Safari 打印窗口下的卡片宽度、文本换行和字段舒展；
   - 降低保存 PDF 时出现拥挤、截断或标题落单的概率。

5. 暗夜 / 阅读模式同步适配
   - 新增打印版式面板、字段按钮、字段标签、打印说明和导出提示均已补暗夜样式；
   - 继续防止新增内容出现浅灰底、白块残留、浅紫字、低对比文字、按钮/标签未继承主题变量等问题；
   - 新增内容暗夜/阅读模式检查继续纳入固定发版体检。

6. Debug / 发版体检增强
   - 新增 `history-print-export-v191` 发版体检项；
   - 新增 `new-content-night-theme-v191` 发版体检项；
   - Debug 历史模块快照新增 `printExportAudit`；
   - 导出字段选择、打印样式、分页防断和新增打印面板暗夜样式均纳入发版检查。

### 本版检查

- 已同步更新页面 title、meta、首页版本标题、版本记录、README、manifest、service-worker。
- 已统一主线 source 标记为 `mainline-v10.5.191`。
- 已检查主脚本语法、重复 ID、旧版本残留和 ZIP 完整性。
- 本版只增强经验库打印、导出字段和暗夜/阅读体检，不修改核心起课算法、落宫规则、断卦逻辑和 JSON 知识库。
