# 小六壬占卜 · v10.5.186 案例库快捷复查与新模块暗夜主题修复版

## v10.5.186 案例库快捷复查与新模块暗夜主题修复版

这一版承接 v10.5.185 的“复盘偏差趋势与案例评分”，继续把案例库往“长期复盘学习工具”推进。同时根据 iPhone/Safari 暗夜模式截图反馈，统一修复 v10.5.179-v10.5.185 新增历史、案例、复盘、评分、导出预览模块的暗夜模式低对比问题。

### 本版新增

1. 新增“案例库快捷复查与学习清单”面板
   - 自动整理当前历史筛选范围内需要回头看的案例；
   - 覆盖低分案例、应期偏差、未填原因归类、待补学习要点、待复盘记录；
   - 清单只用于复盘整理，不影响核心起课、落宫、断卦和知识库逻辑。

2. 快捷复查筛选
   - 支持“全部复查”；
   - 支持“低分案例”；
   - 支持“应期偏差”；
   - 支持“未归类”；
   - 支持“待补学习要点”；
   - 支持“待复盘”。

3. 复查卡片操作
   - 每张复查卡片显示案例评分、等级、场景、时间、问题、落宫与复查原因；
   - 支持一键查看案例；
   - 支持一键打开复盘弹窗，方便补原因归类、学习标签和学习要点；
   - 点击复查原因可写入历史搜索并联动筛选。

4. 新模块暗夜主题修复
   - 修复案例评分卡片在暗夜模式下出现浅灰底、浅紫文字、低对比的问题；
   - 补齐历史标签统计、历史案例库、复盘原因学习、复盘趋势评分、导出范围预览等新增模块的暗夜样式；
   - 统一处理新卡片、标签、评分胶囊、复查原因、导出卡片和复盘学习标签的暗夜背景、文字和边框。

5. Debug / 发版体检增强
   - 新增 `history-quick-review-v186` 发版体检项；
   - 新增 `new-content-night-theme-v186` 发版体检项；
   - Debug 暗夜运行时扫描加入新增历史/案例/复盘/评分模块；
   - 后续新增任何模块、卡片、弹窗、统计面板和导出预览，都必须同步加入暗夜主题覆盖检查。

### 本版检查

- 已同步更新页面 title、meta、首页版本标题、版本记录、README、manifest、service-worker。
- 已统一主线 source 标记为 `mainline-v10.5.186`。
