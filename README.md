## v10.5.225 命运冗余内容清理与日加时寻物知识库完整版

这一版继续基于 v10.5.224 收口，不改起课算法，不改核心断卦逻辑。重点处理命运板块冗余内容，同时把你补充的日加时断与寻物专项 JSON 做成更完整的主线数据源。

### 本版主要更新

1. 命运板块冗余内容清理
   - 移除命运结果页中的落地检查清单、反向验证提示、接下来观察什么、可以怎么追问等后期提示卡。
   - 隐藏“本场景判断重心”模块，命运页不再显示“这次主要看什么 / 判断顺序 / 本课特别提醒 / 命盘实用法”这类教学式内容。
   - 命运页答案摘要改为简洁摘要，只保留命盘摘要、为什么这么看和需要注意。

2. 日加时断知识库升级
   - 日加时断 36 组升级为 `original + explain` 结构。
   - 继续放在“双宫断事解卦”里，只显示当前日宫 + 时宫的一条，不新增顶层大模块。
   - 兼顾婚姻、失物、疾病、行人等场景解释。

3. 寻物专项知识库增强
   - 失物场景补齐单宫寻物口诀、方位、场所、天数、询问对象、物品状态和寻找建议。
   - 增加日加时寻物组合断语。
   - 增加证件、钥匙、手机、钱包、首饰、宠物等特殊物品位置提示。
   - 逐句解释放入折叠区，默认结果页不铺开大全。

4. 显示与字体微调
   - 命运页冗余卡片不再占屏。
   - 命运页答案摘要字体和行距收紧。
   - 新增寻物折叠内容同步适配暗夜模式和移动端。

5. 新增发版体检
   - 新增 `xlr225DestinyDayHourLostAudit`。
   - 检查命运冗余内容清理、日加时完整数据、寻物专项增强、暗夜样式和移动端样式。
