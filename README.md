## v10.5.208 首页起课体验与场景按钮分组合并增强版

这一版继续按照合并版节奏推进，不再只做单点小修。重点优化首页到起课前的整个流程：让用户更容易先选对场景、把占问事项写清楚，再进入数字/时辰/事件起课。

### 本版新增 / 修复

1. 首页快速起课导引
   - 新增“快速起课导引”面板；
   - 用“先定场景 → 再写问题 → 最后看三象”的顺序说明普通问事怎么用；
   - 保留 v10.5.207 的场景×占问事项×占卜结果联动逻辑。

2. 场景按钮分组增强
   - 场景按钮改成更清楚的分组切换；
   - 分组包括常用场景、家庭与生活、人际与关系、学习成长与身心、消费出行与风险；
   - 避免首页一次铺太多按钮，手机/iPad 上更容易选择。

3. 占问事项输入体验增强
   - 占问事项区新增“按当前场景推荐问法”；
   - 可一键把常用日常问题带入输入框；
   - 新增“把占问事项同步到事件起课”，方便直接用问题文字起课。

4. 普通问事快捷模板前置
   - 首页直接显示当前场景最常用的问法；
   - 仍保留原“问题模板”弹窗，适合查看更多模板和自定义模板；
   - 快捷问法只改变输入内容，不修改起课数字、落宫和断卦主逻辑。

5. 暗夜 / 阅读模式同步适配
   - 新增首页导引、场景分组、问法快捷按钮、输入质量提示均已补暗夜样式；
   - 继续避免浅灰底、白块、浅紫字和低对比文字问题。

6. Debug / 发版体检增强
   - 新增 `xlr208HomeInputAudit()`；
   - 新增 `home-input-scene-group-v208` 与 `new-content-night-theme-v208` 发版体检项；
   - 主线 source 统一到 `mainline-v10.5.208`。
