# 小六壬占卜 · v10.5.189 复盘模板应用与经验库筛选版

## v10.5.189 复盘模板应用与经验库筛选版

这一版承接 v10.5.188 的“复盘建议模板与案例经验沉淀”，继续把复盘模板从“可复制”推进到“可应用”。本版重点不是修改核心占卜逻辑，而是让模板可以一键带入复盘备注与学习要点草稿，并新增案例经验库筛选，让高分路径、低分边界、应期经验和高价值样本更容易长期回看。

### 本版新增

1. 复盘模板“应用草稿”
   - 模板卡片新增“应用草稿”按钮；
   - 可把对应模板的复盘填写提示带入复盘备注；
   - 可把经验沉淀说明带入案例学习要点；
   - 保存前仍需用户人工确认，不会自动写入历史。

2. 模板使用次数统计
   - 本地统计每类复盘模板的使用次数；
   - 模板卡片显示“已用 X 次”；
   - 历史记录中心的模板 KPI 增加“模板使用”统计；
   - 使用数据仅保存在本地浏览器，不影响起课和断卦逻辑。

3. 新增“案例经验库筛选”面板
   - 支持全部经验；
   - 支持高分经验；
   - 支持低分边界；
   - 支持应期经验；
   - 支持有学习要点；
   - 支持高价值经验。

4. 经验卡片快捷操作
   - 查看案例；
   - 打开复盘；
   - 复制案例经验；
   - 联动历史搜索。

5. 暗夜 / 阅读模式同步适配
   - 新增经验库面板、经验卡片、经验筛选工具栏、模板使用次数标签均已加入暗夜模式覆盖；
   - 继续防止新增内容出现浅灰底、白块残留、浅紫字、低对比文字、按钮/标签未继承主题变量等问题；
   - 新增内容暗夜/阅读模式检查继续纳入固定发版体检。

6. Debug / 发版体检增强
   - 新增 `history-review-application-v189` 发版体检项；
   - 新增 `history-experience-library-v189` 发版体检项；
   - 新增 `new-content-night-theme-v189` 发版体检项；
   - Debug 历史模块快照新增 `experienceLibraryAudit`；
   - 复盘模板应用、模板使用统计和经验库筛选均纳入发版检查。

- 已同步更新页面 title、meta、首页版本标题、版本记录、README、manifest、service-worker。
- 已统一主线 source 标记为 `mainline-v10.5.189`。

- 本版只增强复盘模板应用、案例经验库筛选和暗夜/阅读体检，不修改核心起课算法、落宫规则、断卦逻辑和 JSON 知识库。
