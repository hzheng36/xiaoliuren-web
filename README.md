## v10.5.170 PDF 明细舒展与打印体验优化版

这一版基于 v10.5.169 继续升级 PDF 报告导出层。按你的反馈，本版跳过“PDF 自定义标题与备注输入”，不增加新的输入弹窗，重点继续优化 PDF 明细拥挤问题，让历史明细和复盘明细更舒展、更适合打印分页、手机阅读和长期归档。

### 本版新增 / 修复

1. PDF 明细舒展增强：
   - 历史明细卡片继续保留报告卡片式视觉，但进一步加大卡片间距、内部 padding 和字段间距。
   - 将历史明细拆成“核心问题 / 判断摘要 / 复盘跟进”三个分区。
   - 占问事项、直接答案、综合判断、三象摘要、最新实际结果和复盘备注分开显示，减少挤在一起的感觉。
2. PDF 复盘明细增强：
   - 将复盘明细拆成“原始占问 / 复盘结论 / 记录信息”三个分区。
   - 实际结果、复盘备注、准确度、准确单位、复盘时间和更新时间更清楚。
   - 长文本继续自动换行，适合保存 PDF 后在手机端慢慢看。
3. PDF 打印体验优化：
   - 继续保留 v10.5.166 已认可的 PDF 视觉风格，不大改配色和整体版式。
   - 保留 v10.5.169 的章节分页、页眉页脚、标题防落单和卡片防断页能力。
   - 打印样式中继续加强卡片、字段块、章节标题和分区标题的分页保护。
4. 跳过自定义标题输入：
   - 本版不新增 PDF 标题/备注输入弹窗。
   - PDF 封面仍使用默认报告标题和自动说明，导出流程保持简单。
   - 封面信息改为强调“明细版式：舒展卡片”和“打印优化：分页防断”。
5. Debug / 发版体检增强：
   - 新增 PDF 明细舒展与打印体验检查。
   - 新增 `ANALYTICS_PDF_DETAIL_COMFORT_MISSING` 警告项。
   - 新增 `PDF 明细舒展与打印体验检查通过` 通过项。

### 使用方法

1. 打开网站。
2. 进入历史分析或历史记录。
3. 在导出区选择：
   - `导出完整 PDF`
   - `只导出图表 PDF`
   - `只导出明细 PDF`
4. 浏览器会打开打印窗口。
5. 在打印选项中选择 `保存为 PDF`。
