## v10.5.206 暗夜历史记录显示紧急修复与合并版回稳版

这一版是紧急回稳修复版，重点处理 v10.5.205 合并升级后在暗夜模式下打开历史记录时，历史列表/历史卡片可能出现文字不可见、区域看起来全空白的问题。

### 本版新增 / 修复

1. 暗夜模式历史记录显示紧急修复
   - 修复历史记录弹窗在暗夜模式下可能出现全空白或低对比的问题；
   - 对历史列表、历史卡片、时间线、折叠面板、批量工具、筛选栏、导出中心和标签胶囊增加统一暗夜可读兜底；
   - 强制修复浅灰底、浅紫字、文字继承不到暗夜变量、按钮/标签低对比等问题。

2. 保留 v10.5.205 合并版功能
   - 保留普通问事重点阅读 / 完整细读；
   - 保留命运页阅读顺序提示；
   - 保留场景规则弹窗分组；
   - 保留移动端导航体验优化。

3. Debug / 发版体检增强
   - 新增 `xlr206HistoryNightAudit()`；
   - 新增 `history-night-readable-v206` 发版体检项；
   - 主线 source 统一到 `mainline-v10.5.206`。

4. 安全边界
   - 本版只修改显示层、暗夜样式和体检接口；
   - 不修改历史数据、复盘数据、起课数字、落宫、断卦逻辑和 JSON 知识库。

### 本版检查

- 已检查主脚本语法。
- 已检查 manifest JSON。

- 已检查上一版 source 残留。

- 已检查 ZIP 完整性。

v10.5.207 可以在确认暗夜历史记录恢复正常后，再继续合并版主线：首页体验整理、起课区输入体验、场景按钮分组和阅读模式细节。
