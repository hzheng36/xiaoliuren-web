# 小六壬个人占卜台 · v9.9.15 显示与桌面模式优化版

这是一个纯前端小六壬个人网页工具，适合部署在 GitHub Pages。

## 本版更新

- 正文阅读字号恢复为更克制的默认大小，保留输入框较大文字。
- 阅读设置新增“阅读文字大小”，可调整解读文字显示大小。
- 右侧快捷按钮统一尺寸，“主页 / 重占 / 电脑”与“展开 / 折叠 / 置顶 / 到底”保持一致。
- 移动端“电脑”模式改为更接近电脑端的宽页面布局，可横向查看桌面版结构。

## 起课规则

本工具采用 **三数连续顺推法**：

1. 第一数从大安起数。
2. 第二数从第一数落宫处继续顺推。
3. 第三数从第二数落宫处继续顺推。

六宫顺序：

```text
大安 → 留连 → 速喜 → 赤口 → 小吉 → 空亡
```

## 本地数据说明

历史记录、复盘、模板、阅读设置等均保存在当前浏览器本地，不联网、不上传。

## 导入导出

- TXT：用于阅读、分享、给后续分析参考。
- 历史 JSON：只迁移历史记录、收藏、复盘和 A/B 数据。
- 模板 JSON：只迁移问题模板。
- 完整备份 JSON：迁移整个个人占卜台，包括历史、复盘、模板和设置。

## 部署到 GitHub Pages

将以下文件放到仓库根目录：

```text
index.html
README.md
```

然后在 GitHub 仓库 Settings → Pages 中选择 main / root 发布。
