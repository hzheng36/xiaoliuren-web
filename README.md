## v10.5.207 场景占问结果联动与日常白话增强版

这一版按照合并版节奏继续推进，不再只做单点小修。重点增强“场景、占问事项、占卜结果”的联动解释，让普通问事结果更像日常对话，而不是只展示术语和模块。

### 本版新增 / 修复

1. 场景 × 占问事项 × 占卜结果联动
   - 新增联动面板，把所选场景、用户填写的占问事项和三象结果放在同一块解释；
   - 强调先看这件事属于什么场景，再看用户到底问什么，最后把本象/中象/末象串成“起点 → 过程 → 结果”；
   - 减少只看单宫吉凶、只看末象、或只看一句结论造成的理解脱节。

2. 白话表达继续日常化
   - 答案摘要新增“生活白话版”；
   - 问事专项解读新增“专项解读先翻成人话”；
   - 三象说明进一步改成更简单的日常表达：现在怎样、过程怎样、最后怎样。

3. 结果页体验合并增强
   - 保留 v10.5.205 的重点阅读 / 完整细读、命运页阅读顺序、场景规则分组和移动端导航优化；
   - 保留 v10.5.206 的暗夜历史记录显示紧急修复；
   - 本版只增强解释层和显示层，不修改起课数字、落宫、断卦主逻辑和 JSON 知识库结构。

4. 暗夜 / 阅读模式同步适配
   - 新增联动面板、生活白话卡片、专项白话说明都已补暗夜样式；
   - 继续避免浅灰底、白块、浅紫字和低对比文字问题。

5. Debug / 发版体检增强
   - 新增 `xlr207LinkageAudit()`；
   - 新增 `result-scene-question-linkage-v207` 与 `new-content-night-theme-v207` 发版体检项；
   - 主线 source 统一到 `mainline-v10.5.207`。
