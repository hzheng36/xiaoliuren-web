# 小六壬占卜 · v9.9.14 多端布局与结果页优化版

这是一个纯前端的小六壬个人占卜台，适合部署到 GitHub Pages。

## 当前版本

**v9.9.14 多端布局与结果页优化版**

本版重点优化多端阅读体验：

- 顶部插图改为多端完整显示，尽量避免手机、平板、电脑端裁切。
- “个人占卜台”和功能按钮分成两排，方便后续继续增加按钮。
- 输入框文字整体放大，手机和平板端更容易阅读。
- 占卜后进入独立结果页，隐藏场景选择、问事输入和起课方式等主页内容。
- 结果页顶部和底部增加“返回主页 / 重新占卜 / 复制结果 / 查看历史”。
- 右侧快捷按钮增加“主页”“重占”“桌面/自适应模式”。
- 保留三数连续顺推法、问题模板、场景规则、历史记录、复盘、历史分析、JSON 备份恢复等功能。

## 起课规则

本工具采用 **三数连续顺推法**：

1. 第一数从“大安”起，顺推落宫。
2. 第二数从第一数落宫处继续起，顺推落宫。
3. 第三数从第二数落宫处继续起，顺推落宫。

六宫顺序：

```text
大安 → 留连 → 速喜 → 赤口 → 小吉 → 空亡
```

## 部署到 GitHub Pages

把以下文件上传到仓库根目录：

```text
index.html
README.md
```

然后在仓库 Settings → Pages 中选择：

```text
Deploy from a branch
Branch: main
Folder: /root
```

等待部署完成后即可访问。

## 数据保存说明

本工具所有数据保存在当前浏览器本地，包括：

- 历史记录
- 收藏
- 复盘时间线
- 自定义模板
- 阅读设置
- JSON 备份恢复数据

如果更换设备或浏览器，请使用“完整备份 JSON”导出并导入恢复。
