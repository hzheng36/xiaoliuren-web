# 小六壬个人占卜台 · v9.9.21 场景扩展与命运排盘版

这是一个纯前端的小六壬网页工具，可部署到 GitHub Pages。数据保存在当前浏览器本地，支持 TXT / JSON 导出导入与 PWA 离线缓存。

## 本版更新

- 新增场景：**寻人、贵人、命运**。
- 命运场景只保留 **时辰起课**，用于填写出生年月日与出生时辰。
- 增强 **六宫排盘** 说明，并在章节导航中新增“排盘”。
- 命运结果页新增 **命运六亲参考盘**，用于出生时间参考、六亲象意参考与自我观察。
- 版本内容更新已按版本号顺序整理，并补充 v9.9.21。

## 起课规则

本工具正式主规则为 **三数连续顺推法**：

1. 第一数从大安起。
2. 第二数从第一数落宫继续起。
3. 第三数从第二数落宫继续起。

时辰起课以用户设备显示的当地时间为准；数字起课与事件起课不依赖地理位置。

## 起课注意事项

- 问题要清晰、具体。
- 一事不二占，一时不起二课。
- 问事时心念专一，不疑不妄。

## 命运场景说明

命运场景以出生时间起象，仅作传统象意、自我观察与人生规划参考，不作为人生定论。
