# 小六壬占卜 · v10.5.224 关键词字号、场景高亮与失物口诀增强版

## v10.5.224 关键词字号、场景高亮与失物口诀增强版

这一版继续沿用 v10.5.223 的普通问事主线结构，不重新增加生活白话、行动清单或大块冗余面板。重点修复两个可见问题，并把用户补充的“日加时断”和“失物寻物口诀”归入现有单宫/双宫断事主线。

### 本版主要更新

1. 关键词标签轻量化
   - 修复占卜结果中“关键词”胶囊标签字体偏大、比正文更抢眼的问题。
   - 关键词前缀改为统一小标签，单个关键词只显示词本身。
   - 移动端进一步压缩字号、内边距和间距。

2. 暗夜模式场景高亮增强
   - 场景按钮同步写入 `active`、`selected` 和 `aria-pressed` 状态。
   - 暗夜模式下选中场景增加更明显的背景、边框、阴影和“已选”提示。
   - 避免选了场景但视觉上看不出来。

3. 日加时断接入双宫断事
   - 接入 36 种“日宫 + 时宫”组合口诀。
   - 显示位置放在“双宫断事解卦”中，只显示当前日宫加时宫的一条断语。
   - 不新增顶层大模块，避免普通问事结果重新变复杂。

4. 失物寻物口诀增强
   - 接入六掌诀寻物核心口诀。
   - 失物场景在“单宫断事解卦”中按末象显示当前口诀、找回概率、方位、场所、天数和寻找建议。
   - 仍提醒结合现实最后出现地点、监控、失物招领和实际搜索。

5. 新增发版体检
   - 新增 `xlr224KeywordSceneLostAudit`；
   - 检查关键词轻量化、场景高亮、日加时断、失物口诀、暗夜样式和移动端样式。
