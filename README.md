# 小六壬占卜 · v10.5.178 历史记录批量管理增强版

## v10.5.178 历史记录批量管理增强版

这一版继续增强历史记录中心，把单条查看/收藏/删除升级为可批量处理。适合历史记录越来越多之后，一次性整理收藏、清理旧记录，或者只导出当前挑选的一批记录。

### 本版新增

1. 历史记录批量选择
   - 每条历史记录卡片新增“选择”勾选框；
   - 新增“全选当前”，会按当前搜索、日期、场景、收藏筛选范围选中；
   - 新增“取消选择”，可一键清空当前勾选。

2. 批量收藏与取消收藏
   - 可对选中记录批量收藏；
   - 可对选中记录批量取消收藏；
   - 若当前结果页正在查看其中一条记录，收藏状态会同步刷新。

3. 批量删除与撤销保护
   - 批量删除前会二次确认；
   - 删除前自动保存上一次批量操作备份；
   - 可点击“撤销上次”恢复到批量操作前的历史记录状态，降低误删风险。

4. 选中记录导出
   - 新增“选中 JSON”；
   - 新增“选中 CSV”；
   - 只导出当前勾选的记录，适合迁移部分历史或做单独分析。

5. Debug 与发版体检
   - 新增 `historyBatchAudit`；
   - 新增 `history-batch-manager-v178` 发版体检项；
   - 检查批量工具栏、状态提示、批量收藏/删除、选中导出和撤销保护是否完整挂载。

### 本版检查

- 已同步更新页面 title、meta、首页版本标题、版本记录、README、manifest、service-worker。
- 已检查主脚本语法。
- 已保留 v10.5.176 iPad Excel 导出兼容体检逻辑。
- 已保留 v10.5.177 历史分析图表钻取与明细联动逻辑。
- 本版只增强历史记录管理和导出层，不修改核心起课算法、落宫规则、断卦逻辑和 JSON 知识库。

### 下一版建议

v10.5.179 可以考虑做“历史记录归档与标签增强版”：给历史记录增加标签/备注/归档状态，用于长期整理大量占卜记录。
